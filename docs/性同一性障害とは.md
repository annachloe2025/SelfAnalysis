# 性同一性障害とは

このサイトで頻繁に登場する **性同一性障害** と、私の状態である **メイレズビアン** という言葉について、最初に簡単に説明しておきます。

「性同一性障害」は広い概念で、**メイレズビアンはその中のかなり特殊な一例** という位置関係です。両者は同じものではありません。

## 三つの軸

人の性のあり方は、ざっくり次の三つの軸で記述できます。

| 軸 | 内容 |
| --- | --- |
| **生物学的性別** | 出生時に身体の特徴で割り当てられた性別。男性 / 女性 |
| **性自認**（gender identity） | 自分自身を心の中でどの性として認識しているか |
| **性的指向**（sexual orientation） | どの性に対して恋愛・性的に惹かれるか |

普通は三つが揃っていることが多いです（例：身体男性、性自認男性、性的指向女性 → ヘテロセクシュアル男性）。

## 性同一性障害（GID）とは

**生物学的性別** と **性自認** が一致しない状態を、医学用語で「性同一性障害（Gender Identity Disorder, GID）」と呼びます（最近は「性別違和」「トランスジェンダー」という呼称が使われることが多くなっています）。

私の場合、生物学的男性として生まれましたが、心の中の性自認は女性です。

## 「メイレズビアン（AMABレズビアン）」とは

ここからが少し珍しい組み合わせです。

| 私の三軸 | 内容 |
| --- | --- |
| 生物学的性別 | **男性** |
| 性自認 | **女性** |
| 性的指向 | **女性** |

つまり、**「中身は女性で、女性が好きなレズビアン」だが、身体は男性** という組み合わせです。

これを表す呼び方には、いくつかあります：

- **AMAB レズビアン** — AMAB は "Assigned Male At Birth"（出生時に男性と割り当てられた）の略
- **メイレズビアン** — 「Male（身体）× レズビアン（中身）」の合成語。私はこれを好んで使っています
- **トランス女性レズビアン** — トランスジェンダー女性で、性的指向が女性

外見からは「男性で、女性が好き」（=ヘテロセクシュアル男性）と区別がつかないため、本人にも周囲にも気づかれにくい組み合わせです。私自身、47歳まで、自分がそうだと言語化できていませんでした。
