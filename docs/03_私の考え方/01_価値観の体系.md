# 価値観の体系

私の価値観は次の三層構造になっている。

## 階層構造

### 最上位：全体の幸福の最大化（功利主義的原理）

- できるだけ多くの人の幸福を、できるだけ大きくしたい
- 個人の利益より全体の最適化を志向する
- ケーキは一番小さいものを最初に食べる（[認知的傍証](../05_根拠とエピソード/03_認知的傍証.md)）

### 中位：論理的整合性と本質の追求

- 物事の道理や仕組みの理解を重視する
- 根拠のない正当化を嫌う
- 客観的な自己評価を重んじる

### 基底：人間の尊厳の尊重（ヒューマニズム）

- いじめ・差別・抑圧への強い拒否感
- 弱者を踏みつける構造への怒り

これは私の[特性 / 派生的特性のマップ](../02_私の特性/06_派生的特性のマップ.md)で論じた「派生1：功利主義者・ヒューマニスト・理想主義者」の中身だ。

## 重視すること／重視しないこと

| 重視すること | 重視しないこと |
| --- | --- |
| 全体の幸福・合理性 | 地位・肩書き・権威 |
| 物事の本質と道理 | 他者からの評価・賞賛 |
| 人の尊厳と公正さ | 競争での勝利・優越感 |
| 自己理解・自己分析 | SNSでの承認・注目 |
| 成長・能力の向上 | 所属集団での立場 |
| 整合性・矛盾の解消 | 派閥・空気・忖度 |
| 知識共有のフラットさ | マウンティング・権威主義 |

これは具体的な選択場面で繰り返し現れる。

## 価値観の起源

各層の価値観がどこから来たか、私の自己理解に基づいて整理する。

### なぜ「全体の幸福の最大化」を最上位にしたか

- **承認欲求がない**ので、自分個人の取り分にこだわる動機がない
- **HSP**で他者の痛みに反応するので、誰かが苦しんでいる状態は自分の不快を生む
- 結果、最大化対象は「自分」より「全体」の方が、自分の不快を最小化する解になる
- 小学生時にマザー・テレサに憧れたのは、自分のロジックの祖型をすでに体現している人物だったから

### なぜ「論理的整合性」を中位にしたか

- **承認欲求がない**ので、「みんながそう言っている」「権威がそう言っている」では納得できない
- **HSP × 開放性高**で、表面で済ませず構造まで掘る癖がある
- 内発的動機が乏しいので、行動の根拠を**自分で再構築する**必要があった
- これが整合性を最上位の判断基準にする習慣を作った

### なぜ「人間の尊厳」が基底にあるか

- **HSP**でいじめ場面の被害者に強く同一化する
- 02-5-3 の給食強制事件のような構造（同調圧力 × 権威 × 個人）への怒りが、社会観の原点になっている
- 戦争・いじめ・嫁姑問題が「他人事」ではなく自分の不快として刺さる
- 結果、人間の尊厳を踏みつける構造への拒否感が、すべての判断の最低限の制約条件になる

## 私が嫌うこと

価値観はネガティブな側でも明確に出る。私は次のものを強く嫌う：

- **マウンティング** — 知識共有を立場確立の手段にする態度
- **理由のない正当化** — 「なんとなく」「みんながそうだから」の根拠
- **承認欲求由来の暴力** — 31歳の市場の理不尽な馬鹿にしてくる人々
- **構造的搾取** — 善意で始まった管理が支配に変わる構図
- **見えない攻撃** — 嫁姑関係に代表される、明示されない加害
- **権威主義** — 偉い人＝お金持ち、という暗黙の前提
- **エモい的な言語化能力の縮退** — [MyConsiderations 該当記事](https://annachloe2025.github.io/MyConsiderations/言語/2026-04-25_「エモい」と言語化能力の縮退/)
- **善意の管理** — [善意の管理が支配に変わるとき](https://annachloe2025.github.io/MyConsiderations/社会/2026-04-09_善意の管理が支配に変わるとき/)

## 私が大事にしている習慣

価値観を日常に落とすと、私は次の習慣を維持している：

- **20歳から「会話した相手が会話前より賢くなるように」話している** — マウンティングではなく贈与
- **整合性の検出は手を抜かない** — ChatGPTの言い方のずらしを毎回指摘する
- **黒と白の服を着る** — 目立とうという発想を消すための物理的設定
- **人間関係比重を最小化した職業を選ぶ** — 朝の新聞配達、Uber Eats
- **オーディブルを聴き続ける** — 起きている時間のほぼ全て
- **エッセイ・考察ブログ・このサイトを継続更新する** — 思考過程を残す

## 価値観の変化（戦略的撤退）

47歳になるまで、私の価値観の最上位は「全体の幸福の最大化」だった。だが、47歳の自己理解後に **「世界平和は構造的に実現不能」** と判定した。

理由：
- 人類の99%以上は承認欲求（4段階目）を本能として持つ
- 4段階目本能は、合理ではなくポジション取りで動く
- 戦争・いじめ・嫁姑問題は「悪意」ではなく「数百万年の本能」が動いている
- 数百年の文明では追いつかない速度差

この判定を受けて私が選んだのが **戦略的撤退** だ：

- 自分の幸せより世界平和を望みたいけれど、考えると100%不幸になるので考えない
- 代わりに、自分のツボに最適化した小説・動画を作ることに集中する
- 自己満足を極めることが、結果的に同じ構造を持つ少数派への最大の贈り物になる

> 自分の幸せより世界平和を望みたいけど、それを考えると100%不幸になるので考えない。
> （`MyConsiderations/docs/哲学/2026-05-02_執筆動機の根.md`）

> 自己満足を極めることが、結果的に他者への最大の贈り物になる。
> （同上）

これは価値観の放棄ではなく、達成不能性の認識を踏まえた戦略のリビジョンだ。最上位は依然「全体の幸福の最大化」だが、自分の影響力で動かせる範囲を「世界全体」から「同じ構造を持つ少数派」へ縮小した。
