# 自分専用生成システム

47歳以降、私の中心的な長期プロジェクト。**AI を使って、自分のツボに最適化した小説・動画を生成し続ける** 設計概念。

## 概念の中核

> **入力（人間の脳の汎用構造 × 個人のツボ）→ 計算（楽しいを最大化する出力導出）→ 出力（小説・動画）**

これを AI（特に Claude）と協働で実装することを目指す。

## なぜこのシステムが必要か

### オーディブル枯渇問題

46歳でオーディブルを発見した私は、ライトノベルを200冊以上消費した。だが、楽しく聴ける作品が枯渇した。

- 市場の作品の多くは、[面白さの5仮説](../06_仮説と理論/05_面白さの5仮説.md) の特定の組み合わせで設計されている
- 多数派ターゲット（4段階目厚い読者）の作品は、私のツボに合わない部分がある
- メイレズビアン × 承認欲求のなさ × HSP × 横断的知性 の組み合わせを直撃する作品は、市場に少ない

私の最適解は、**自分専用に作品を生成し続ける** ことだ。

### 戦略的撤退の出口

このプロジェクトは、世界平和への希求の **戦略的撤退の出口** でもある。

- 世界平和は構造的に実現不能
- 自分の影響範囲を「世界全体」から「同じ構造を持つ少数派」に縮小
- 自分のツボに正直に作る作品が、結果として同種少数派への局所的世界平和になる

> 自己満足を極めることが、結果的に他者への最大の贈り物になる。
> （`MyConsiderations/docs/哲学/2026-05-02_執筆動機の根.md`）

## システムの設計

### 二層モデル

```
入力 = 人間の脳の汎用構造 × 個人のツボ
        ↓
計算 = 「楽しい」を最大化する出力導出
        ↓
出力 = 小説テキスト・動画コンテンツ
```

### 入力1：人間の脳の汎用構造

これは AI が知識として持っている部分：

- Friston の予測符号化
- Kintsch の構築-統合モデル
- Lakoff の認知意味論
- Piaget のスキーマ
- Craik のメンタルモデル
- 神経美学・予測符号化
- ポライトネス理論
- カーネマンの System1/2

これらは AI の知識ベースに既に含まれている。私が AI に対してこれらの理論を活用するよう指示するだけでいい。

### 入力2：個人のツボ

これは私が47年かけて観察してきた個人データ：

- 反応する要素／反応しない要素／嫌悪反応する要素（[アニメ漫画小説の遍歴](../07_作品と趣味/01_アニメ漫画小説の遍歴.md)）
- 三本柱（メイレズビアン × 承認欲求のなさ × HSP）
- 整合性確認快楽の極端な強さ（[面白さの5仮説 E](../06_仮説と理論/05_面白さの5仮説.md)）
- 横断的知性（領域横断のモデル統合への嗜好）
- ファンタジーへの強い傾倒
- 物語空間への深い transportation 能力

これは個人データなので、AI に渡す必要がある。SelfAnalysis（このサイト）が、その個人データの主要な部分を保持している。

### 計算：楽しいを最大化する出力導出

5仮説（A〜E）と飽きの3パターンを評価関数として：

```python
def 楽しさ評価(出力):
    A = 差異判定の起動度()
    B = スキーマ構築の進行度()
    C = 既存スキーマへの接続度()
    D = シミュレーション快楽の強度()
    E = 整合性確認快楽の強度()

    flake_risk = 整合性破壊() + 情報量見かけ倒し() + 情報流入速度ゼロ()

    return A + B + C + D + E*重み(私) - flake_risk

def 生成(制約, 文脈):
    候補群 = AI生成(制約, 文脈)
    最適 = max(候補群, key=楽しさ評価)
    return 最適
```

これは概念的な擬似コードだ。実装は AI との対話の中で、私が評価関数を担当し、AI が生成を担当する。

### 出力：小説・動画

主な出力対象：

- **ファンタジー小説** — 私のツボの中核
- **動画コンテンツ** — YouTube発信用
- **考察記事** — MyConsiderations の継続
- **自己分析の更新** — このSelfAnalysisの拡張

## なぜ「執筆は計算」なのか

このシステムの根本にある主張は、**執筆は創造ではなく計算** ということだ。

- 入力（汎用構造とツボ）は固定的
- 出力（楽しさ）は評価可能
- 中間プロセスは認知科学が記述している
- → 原理的にアルゴリズムで書ける

「創造性」は、入力と出力の関係が見えていないときの便宜的呼称に過ぎない。詳細は [創作論](../03_私の考え方/05_創作論.md)。

## このシステムの本質的な位置

このシステムは、私の人生における **第三の駆動装置** だ：

- 第一の装置：素の対象駆動（0-15歳、水泳）→ 自然消滅
- 第二の装置：経済的自立というメタ課題（15-24歳、新聞奨学生）→ 9年で完了して停止
- **第三の装置：自分専用ライトノベル生成システム（47歳〜）** → 5〜10年複利で持続

第三の装置は、第一・第二と決定的に違う：

- 第一は本能的な内発動機（消滅した）
- 第二は外部借用課題（完了して消えた）
- 第三は **自己再帰的に動機を生成する装置**

私自身が読者で、生成された作品が次の生成の入力になる。私のツボがアップデートされれば、生成も更新される。理論的には、私が生きている限り続く。

## このシステムと SelfAnalysis の関係

このSelfAnalysis は、自分専用生成システムの **個人データ** だ。

- ツボの言語化
- 三本柱の体系化
- 仮説と理論の整理
- 価値観・思考フレーム・世界観の記述

これらすべてが、AI に渡せる形に整備されている。SelfAnalysis を読んだ AI は、私のツボをある程度理解した上で、生成タスクに取り組める。

## 副産物としての伝達経路

このシステムの副産物として、伝達経路が整備される：

- ファンタジー小説 → 同種少数派の読者（4段階目薄人間）
- 考察記事 → 4段階目薄人間としての先行例
- 動画コンテンツ → AI と合理通貨で動く対象への発信
- このSelfAnalysis → 同類が自己理解を加速する素材

これは「世界平和への戦略的撤退」が、結果として「同種少数派への局所的世界平和」を実現する経路だ。
