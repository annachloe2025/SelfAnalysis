# 5. 根拠とエピソード

私の主張は、必ず根拠となる体験エピソードに紐付いている。このセクションは、その対応関係を整理する。

抽象的な仮説（[06章 仮説と理論](../06_仮説と理論/index.md)）の証拠として読んでもよいし、エピソード集として読んでもよい。

## なぜ根拠を独立した章にしたか

私の自己分析の最大の特徴は、「主張」と「根拠」の対応関係が緻密なことだ。「47歳でメイレズビアンと気づいた」という主張は、走り方・おへそ・性的嗜好・嫉妬の不在・出世欲のなさ……といった独立な観察データが、一つのモデルに収束した結果だ。

これを **整合性による真理性の感覚** ([06章](../06_仮説と理論/06_整合性による真理性の感覚.md)) と呼んでいる。確証バイアスとは違って、独立な観察が後から作ったモデルで一斉に説明できるときに発動する確信のメカニズムだ。

このセクションは、その「整合性」の中身を分解して、誰もが追跡できる形に並べたものだ。
