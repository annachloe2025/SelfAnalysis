# 承認欲求10項目検証

マズローの承認欲求は、細かく見れば10項目に分解できる。一つずつ自己照合した結果を整理する。

これはエッセイ原本の `04-1-3` から `04-1-12` までの13ファイルに対応する内容を、要約・統合したもの。

## 結論

**全10項目に対して「ほぼ存在しない／弱い／欠けている」という結論**。

| # | 項目 | 自己診断 |
| --- | --- | --- |
| 1 | 評価や賞賛を求める欲求 | 非常に弱い、あるいはほとんど存在しない |
| 2 | 地位や立場に関する欲求 | ほとんど存在しない |
| 3 | 肩書きや称号への欲求 | 明確に存在しない |
| 4 | 他者との比較や優越 | 弱い／欠けている |
| 5 | 注目や可視性を求める欲求 | ほとんど存在しない |
| 6 | 集団内評価への欲求 | ほとんど存在しないと断言 |
| 7 | 自分を肯定したい欲求 | 非常に弱い、あるいは欠けている |
| 8 | 自信や有能感を保ちたい欲求 | 承認欲求としては存在しない（成長欲求は別） |
| 9 | 自分の選択を正当化したい欲求 | 理由なき正当化欲求は存在しない |
| 10 | アイデンティティを守ろうとする欲求 | 守るべきアイデンティティが47歳まで形成されていなかった |

## 他者承認の6項目

### 1. 評価や賞賛を求める欲求

**結論：非常に弱い、あるいはほとんど存在しない。**

- 褒められれば嬉しい（基底レベルの感情）
- 評価されたくて行動することは意識的にしたことがない
- 知識共有を「マウンティング」と誤解されることを繰り返し経験 → 人と話さない生活へ

私は「褒められるために何かをする」発想が浮かばない。誰かが「やった！」と喜ぶ場面を見ても、それを真似したいと思わない。

### 2. 地位や立場に関する欲求

**結論：ほとんど存在しない。**

- 20代から「出世して偉くなりたい」感覚が分からなかった
- 出世＝責任が増えるだけ
- 課長になれば責任が単純に6倍、給料は1.3倍 → コスパが悪い
- 霊長類段階の本能ではないかと仮説

> 出世するということは、より大きな責任を負うことを意味する。単純に考えても、責任は六倍になる。
> （`04-1-4_地位や立場に関する欲求について.md`）

### 3. 肩書きや称号への欲求

**結論：明確に存在しない。**

- 22歳で新聞販売店店長として「使う側／使われる側」構造を内側から自覚
- 「自分は一生、使われる側でいい」と意識的に決断
- 肩書を取る合理的理由が見当たらなかった

> 自分は一生、使われる側でいい。
> （`04-1-5_肩書きや称号への欲求について.md`）

これは断言できる項目。

### 4. 他者との比較や優越

**結論：弱い／欠けている。**

- 麻雀で常に三万点前後の二位タイプ、大負けが少ない
- 勝ちたい人や辛そうな人がいると無意識に手を抜く
- 他者と比較して優位に立ちたいという動機が発動しない

> 自分は勝ちに執着できないタイプの人間なのだ。
> （`04-1-6_他者との比較や優越に関する欲求について.md`）

### 5. 注目や可視性を求める欲求

**結論：ほとんど存在しない。**

- 極力目立ちたくない
- 黒か白の服を好む
- 「目立とう」という発想自体が浮かんでこない

> 「目立とう」「注目されよう」という発想自体が、自分の中にはほとんど浮かんでこない。
> （`04-1-7_注目や可視性を求める欲求について.md`）

### 6. 集団内評価への欲求

**結論：ほとんど存在しないと断言。**

- 全体主義的に「役割は果たすべき」とは思うが、評価されたい欲求由来ではない
- SNSの「いいね」が大量についた経験を **「気持ち悪い」** と表現
- 意見への賛否や異論を求めていたのに無言の反応が来ることに違和感

> 気持ち悪いと感じた。私は自分の意見に対する賛否や、異なる視点からの意見を聞きたかったのであって、無言の反応を集めたかったわけではない。
> （`04-1-8_集団の中での評価に関する欲求について.md`）

## 自己承認の4項目

### 7. 自分を肯定したい欲求

**結論：非常に弱い、あるいは欠けている。**

- 20代は自分を完全にダメ人間だと思っていた
- 「これでいいのだ」という指針を本能的に得られなかった
- 行動量と自己評価が噛み合わなかった

> 実際の行動量と、自己評価がまったく噛み合っていなかった。
> （`04-1-9_自己承認・自尊心に関する欲求について.md`）

承認欲求がないことの **負の側面** がここに集中する。自己肯定の基底が弱く、20代のうつの一因になった。

### 8. 自信や有能感を保ちたい欲求

**結論：承認欲求としては「ほとんど存在しない」／成長欲求としては「強い」。**

ここは **重要な切り分け** が必要だ：

| 区別 | 私の状態 |
| --- | --- |
| 「根拠なく自分はできると思いたい」（=承認欲求） | ない |
| 「能力を高めたい・成長したい」（=成長欲求） | かなり強い |

> 自信を「保ちたい」という承認欲求は、私にはほとんどない。能力を「高めたい」という成長欲求は、かなり強い。
> （`04-1-10_自信や有能感を保ちたいという欲求について.md`）

「承認欲求がない＝向上心がない」と誤解されやすいが、別の話。私は学習・成長への動機は明確に持つ。

### 9. 自分の選択を正当化したい欲求

**結論：理由のない正当化欲求は「ない」／間違いを減らしたい欲求は「ある」。**

ここも切り分けが必要：

| 区別 | 私の状態 |
| --- | --- |
| 「理由のない正当化」 | 嫌い、しない |
| 「間違いを修正する」 | する、間違いを放置できない |
| 「間違うことが平気」 | NO、間違いは修正したい |

> 理由のない正当化という行為が、とても嫌いだ。
> （`04-1-11_自分の選択を正当化したいという欲求について.md`）

私は「自分の選択は正しかった」と無条件に主張する欲求がない。むしろ、選択の根拠が崩れたら、選択を修正する。これは合理回路の特徴だ。

### 10. アイデンティティを守ろうとする欲求

**結論：守る欲求が弱かったのではなく、守るべきアイデンティティそのものが47歳まで形成されていなかった。**

最重要項目。

私が47年間アイデンティティが希薄だった。22歳のカレーランチ事件（[認知的傍証](03_認知的傍証.md)）に象徴される、自分の主観の不在。47歳でメイレズビアンと気づいて、初めて：

> 自分の中に魂が入った。
> （`04-1-12_アイデンティティを守ろうとする欲求について.md`）

> 守るべきアイデンティティそのものが長い間形成されていなかった。
> （同上）

普通の人は10〜20代でアイデンティティが形成されて、それを守る欲求が発動する。私の場合、形成自体が起きていなかった。

なぜか：
- 性自認・性的指向が長らく言語化できなかった
- 配偶ゴール不成立 → 地位獲得本能不発動 → アイデンティティ形成の本能的要請がない
- 4段階目薄人間にとって、アイデンティティ形成は能動的な選択を要する

これが47歳の自覚で根本的に変わった。

## 「ない」の意味の再確認

この10項目すべてに「ない」「弱い」「欠けている」と判定したが、混同を避けるため、何が「ある」かを再確認する：

| ある欲求 | 内容 |
| --- | --- |
| 1段階目（生理的） | 普通にある |
| 2段階目（安全） | 普通にある |
| 3段階目（社会的） | 弱くある（友人がいてほしい程度） |
| **5段階目（自己実現）** | 強くある（能力を高めたい、知的好奇心） |
| 自己肯定の基底 | 弱い（4段階目の不在の副作用） |
| 誤り訂正欲求 | 強くある |
| 整合性欲求 | 強くある |
| 全体最適への動機 | 強くある（功利主義的） |

承認欲求（4段階目）の不在は、他の欲求の存否とは独立した話だ。

## 重要：「成長欲求」と「自己承認」の切り分け

このページで何度も繰り返したが、最も誤解されやすい区別を再強調する：

### 同じに見えるが違うもの

- **「自分はできるはず」と保ちたい** = 承認欲求（自己承認） = **ない**
- **「能力を高めたい・伸ばしたい」** = 成長欲求 = **ある**

承認欲求がない人は無気力ではない。能力開発への動機は別チャンネルで動いている。

### 行動の違い

- 承認欲求由来：「すごいねって言われたいから資格を取る」
- 成長欲求由来：「この知識領域を理解したいから本を読む」

私の学習行動は後者だ。資格・肩書を取る動機は薄いが、知識を増やす動機は強い。
