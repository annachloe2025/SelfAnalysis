# EP-024 アニメを「制作者のように」見る癖がある

**時期:** 大人（継続）

**内容:**
私はアニメ制作者のようにアニメを楽しむ傾向がある。このアニメで伝えたいテーマは何かを考えたり、このシーンはどういう印象を受けるかを感じたり、その表現方法がどうだからそういう印象を受けるのか考えたり、世界観の構築の理由や世界観から感じるイメージを考えたり、キャラのセリフから感じる情動とその表現方法を考えながら見る。多くの人は作品の一部分や印象的な場面を中心に楽しんでいるように感じる。アニメを話題づくりの手段として利用している人も少なくない。

## 関係する特性

**HSP**、**承認欲求の不在**

## 考察（原本より）

「自身に承認欲求がないと知ってからは、承認欲求を満たすためのアニメの利用法とかもあるかもしれないと思い始めた。また、自身がHSPと気が付いてからは、多くの人はアニメで視聴者に伝えようとしている演出はHSPである自分ほど、多くの人は感じ取れていないのではないかと思うようになった。つまり、承認欲求がないので、承認欲求目的でのアニメの活用がない、HSPの影響でより多く影響をアニメから受ける、の理由で普通の人より私は、アニメが好きになりやすい体質なのである。」
