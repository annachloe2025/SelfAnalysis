# EP-019 専門学校時代、推理小説で「頭のいい人はこういう思考をする」と感じた

**時期:** 専門学校時代（10代後半〜20代前半）

**内容:**
専門学校に進学してからは、友人の勧めで推理小説を読むようになった。森博嗣や京極夏彦の作品が好きで、よく手に取っていた。「頭のいい人はこういう思考をするのか」と感じ、自分でもいろいろなことを学びたいと思うようになった。

**タグ:** `#20代` `#物語愛` `#思考傾向` `#結果寄り`

**出典:** 自身のエッセイ/02_自分の特性/02-2_アニメ、漫画、小説が好き/02-2-1_私の遍歴（アニメ・漫画・小説）.md

## 関係する特性

**HSP**、**承認欲求の不在**

## 考察（原本より）

推理小説に触れて「頭のいい人はこういう思考をするのか」と感じ、自分でも学びたいと思うようになった。これは「能力を高めたい」という成長欲求であり、承認欲求（褒められたい・有能と思われたい）とは別系統だと本人が整理している（04-1-10）。HSP由来の深い処理と相性が良い読書ジャンルでもある。
