# EP-025 格闘ゲームで一貫して女性キャラを選んだ

**時期:** 高校生の頃

**内容:**
高校生の頃、格闘ゲームをよく遊んでいた。友人とゲームセンターに行くことも多く、そのとき私はプレイキャラとして女性キャラクターを選ぶことが多かった。女性キャラばかり使っていると周囲の目が少し気になり、意識して男性キャラも使うようにしていたが、本当は女性キャラを操作するほうがずっと楽しかった。
