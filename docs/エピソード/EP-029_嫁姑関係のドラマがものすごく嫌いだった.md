# EP-029 嫁姑関係のドラマがものすごく嫌いだった

**時期:** 子供の頃

**内容:**
子供の頃、ドラマなどでよく目にした嫁姑関係がものすごく嫌いだった。そこに映るやり取りはひどく醜いものに見えた。

## 関係する特性

**承認欲求の不在**、**性同一性障害**

## 考察（原本より）

「私は、嫁姑関係の対立はまさに承認欲求の発露だと思う。そこで起こる衝突の多くは、「どちらがこの家庭や息子にとって重要な存在か」という承認ポジションの取り合いに近い。しかし私は承認欲求がほとんどないため、その感覚がまったく理解できなかった。」さらに「私の性自認が女性で、性的指向も女性に向いていることから、女性のあり方に対する見方が無意識に厳しくなっていた」という性同一性障害由来の女性嫌悪も加算されている。
