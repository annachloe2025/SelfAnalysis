# EP-036 麻雀で勝ちに執着する人を見ると無意識に手を抜く

**時期:** 高校生の頃〜

**内容:**
高校生の頃から麻雀をよく打っていた。アルバイト先のラーメン屋の店長が麻雀好きで、バイト仲間とよく卓を囲んでいた。麻雀を続けていく中で、自分の傾向がはっきりした。私は大体三万点前後の二位になることが非常に多く、トップになっても僅差のトップ、大負けすることは少ない、というタイプ。あまりにも勝ちたい雰囲気を強く出している人や、負けが続いて辛そうな人がいると、無意識のうちに手を抜いてしまうことがあった。連勝中も淡々と勝ち続けられる人がいるが、自分はできなかった。対戦型のゲームや麻雀は意識的に避けるようになった。

## 関係する特性

**承認欲求の不在**、**HSP**

## 考察（原本より）

「自分には承認欲求がほとんどないのだと理解したとき、ようやく腑に落ちた。勝敗に強く執着できないのは、この比較・優越に関する承認欲求が弱い、あるいは欠けているからなのだろうと、今では考えている。」加えて「あまりにも勝ちたい雰囲気を強く出している人や、負けが続いて明らかに辛そうな人がいると、無意識のうちに手を抜いてしまう」というHSP的な共感反応も働いている。
