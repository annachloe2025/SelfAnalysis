# EP-013 マザー・テレサのように生きるべきだと本気で思っていた

**時期:** 小学生の頃

**内容:**
小学生の頃、尊敬する人はマザー・テレサで、「人はみんなマザー・テレサのように生きるべきだ」と本気で思っていた。どうすれば世界が平和になるのか、いじめがなくなるのかをよく考えていた。

## 関係する特性

**性同一性障害**、**承認欲求の不在**

## 考察（原本より）

「私がこうした発想を持つようになった背景には、自分の性のあり方と、承認欲求がほとんどないという性質が大きく関係していると思う。」「普通の人が持つような「自分の内側から湧き出る人生の目的」が、私にはなかった。だからこそ、生きる意味を必死で考え、頭の中で一から組み立てていくしかなかった。」
