# EP-030 給食を残せない男子に女子が無理に食べさせる場面が嫌だった

**時期:** 学校時代（おそらく小学校）

**内容:**
学校では、いじめられている男子に対して、周囲の女子が「給食を残してはいけない」と言って無理に食べさせようとする場面があった。男子は吐きそうになりながら嫌いな給食を口に運んでいた。それを女子が「残しちゃだめ」と言って食べさせ、先生もそれを容認していた。この状況がものすごく気持ち悪く、強い嫌悪感を覚えた。
