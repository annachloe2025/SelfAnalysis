# EP-010 中学三年生のとき、クラスメートに突然殴られた

**時期:** 中学三年生

**内容:**
特に不良というわけでもないクラスメートに、突然教室で殴られた。理由は今となっては覚えていない。一つ強く印象に残ったのは、殴った彼の表情がどこか満たされたように見えたこと。私は「この人にはもう近づかない方がいい」と直感的に思った。彼はおそらく意味もなく人を殴ることで何かを満たしたのだろう。その「満たされる感じ」を心底気持ち悪いものだと感じた。

## 関係する特性

## 考察（原本より）

「殴った彼の表情がどこか満たされたように見えたのだ。」「彼は、おそらく意味もなく人を殴ることで、何かを満たしたのだろう。」「これは承認欲求の有無や在り方の違いだったのではないかと思っている。他人に暴力を向けることで、自分の存在感や優位性を確認する。そうした感覚が、私にはどうしても理解できなかった。」
