# EP-023 46歳頃、オーディブルで一日中楽しい気分で過ごせるようになった

**時期:** 46歳頃

**内容:**
46歳頃からオーディブルの存在を知り、聴きながら仕事や生活をするようになった。これは驚くほど効果があり、一日中楽しい気分で過ごせるようになった。数か月間ほとんど途切れることなく心が満たされた状態が続いた。Kindleの読み上げ機能でも同じような感覚を得られた。ただ、心から面白いと感じる作品でオーディブル化されているものはそれほど多くなく、また常に「楽しい状態」でいると新しいことを学ぶ意欲が薄れる副作用もある。

## 考察（原本より）

HSP由来の没入しやすさが、オーディブルでの長時間の幸福感をもたらした。同時に、常に没入状態にいると他の活動への意欲が下がる副作用もある。
