# EP-041 煙草とパチンコをかなり努力してやめた

**時期:** 不明（おそらく20代〜30代）

**内容:**
ドーパミン系の欲求は顕著にあった（煙草、パチンコ）。どちらも努力してやめた。自分は欲求を合理性で処理する。内から出てきた欲求に対して「これは合理的か」と判断し、合理的でなければ時間をかけてでも手放す。
