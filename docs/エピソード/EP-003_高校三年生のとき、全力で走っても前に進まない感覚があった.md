# EP-003 高校三年生のとき、全力で走っても前に進まない感覚があった

**時期:** 高校三年生

**内容:**
高校三年生頃に全力で走った際、思ったほど前に進まないという不思議な感覚を覚えた。現在の認識では、骨盤が後傾し、結果として効率の悪い走り方になっていた可能性が高いと考えている。当時テレビで見かけたロンドンブーツの田村淳の走り方に近かったのではないかと思う。
