# EP-003 高校三年生のとき、全力で走っても前に進まない感覚があった

**時期:** 高校三年生

**内容:**
高校三年生頃に全力で走った際、思ったほど前に進まないという不思議な感覚を覚えた。

後年、長く便秘症に悩まされ、その原因を究明していく過程で、自分の骨盤が後傾していることに気がついた。いわゆる猫背になっていて、その状態で走っていたために効率の悪い走り方になっていたのだと現在は考えている。

なぜ骨盤後傾・猫背になったのかと聞かれると困るが、とにかく体が硬かったことが大きな原因だと思っている。高校生の頃は姿勢などに気をつけていなかったので、その時期に猫背の状態で筋肉が固まってしまった、ということはあり得る。

## 関係する特性

3つの特性のいずれかと直接的に紐づける根拠は薄い。骨盤後傾・猫背・筋肉の硬さといった **体質や姿勢の問題** で説明できる範囲が大きい。

ただし、次の点は仮説として置いておける:

- HSPの「神経が敏感すぎる」性質と、筋肉が硬直しやすい体質との関係は、可能性としてあり得るかもしれない
- 私自身、筋肉痛になりやすかったのは事実だが、それがHSP由来かどうかは確証がない

EP-002（女の子みたいと言われていた走り方）とは原因の系統が違うエピソードとして扱うのが正確だと考えている。EP-002は性自認との関連が考えやすい一方、こちらは姿勢・体質寄りの話で、特性との因果はそこまで明確に引けない。

## 考察

便秘症 → 原因究明 → 骨盤後傾・猫背の発見、という事後的な気づきの過程で、高校時代の「走っても進まない感覚」の答えがあとから出てきた、というのが時系列。

仮にHSP的な神経の敏感さで筋肉が硬直しやすい体質だったとしても、それが直接「3つの特性のうちのHSP由来である」と断定できるかは微妙で、ここでは **3特性との因果は確定させずに、体質・姿勢の話として記録しておく** ことにする。
