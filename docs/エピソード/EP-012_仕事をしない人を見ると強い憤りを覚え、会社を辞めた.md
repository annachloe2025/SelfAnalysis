# EP-012 仕事をしない人を見ると強い憤りを覚え、会社を辞めた

**時期:** 20代〜30代

**内容:**
人生の戦術として、会社などの組織の中でできるだけ仕事をせずに給料をもらおうとする人がいる。以前の私は、なぜそういう行動を取るのかがまったく理解できず、強い憤りを覚えていた。仕事をしない人が増えれば会社の業績は悪化し、自分たちの給料も下がる、と思っていたから。「おかしな世の中だ」「やっていられない」と感じて会社を辞め、同じことを繰り返していた。

## 関係する特性

**承認欲求の不在**

## 考察（原本より）

「自分には承認欲求がほとんどないのだと理解してから、ようやくその行動が少し見えるようになった。多くの人は、承認欲求に従って行動している。職場の人間関係の中で優位な立場を作り、仕事をあまりしなくても済む位置に収まる。そうすることで、「得をしている」「上に立っている」という感覚を得る。それ自体が、ある種の承認欲求を満たしているのだと思う。当時の私は、このズレを理解できていなかった。その結果、ひとりで憤り、「おかしな世の中だ」「やっていられない」と感じ、会社を辞める。そして、同じことを繰り返していた。」
