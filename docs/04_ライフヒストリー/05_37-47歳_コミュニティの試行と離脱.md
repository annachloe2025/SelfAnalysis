# 37-47歳：コミュニティの試行と離脱（10年）

37歳から47歳までの10年は、社会の本流から離脱した後、**「コミュニティに入った方がいいのではないか」** と試行錯誤した10年だ。結論は、すべて失敗。

## 概況

| 年齢 | 出来事 |
| --- | --- |
| 37歳前後 | 岡田斗司夫オンラインサロン参加→離脱 |
| 40歳前 | 地球防衛軍5（1か月で300時間） |
| 40歳前後 | ドラゴンズドグマオンライン（3か月） |
| 42-43歳 | コロナ期。VRChat に深く参加。Blender 集中習得、Booth 出品 |
| 直近数年 | キノクエスト（YouTuber）の動画編集バイト→離脱 |
| 直近 | FF11復帰。高校友人と1年半で5000時間プレイ |
| 46歳 | オーディブル開始。ライトノベル200冊以上消費 |

## なぜコミュニティを試したか

37歳で社会の本流から離脱した後、私は「人間関係比重を最小化した職業（新聞配達）」で安定した。だが、別の問題が現れた。

- 安定はしたが、孤立感はある
- 社会から離れているが、誰かと話したい欲求はある（弱くはあるが社会的欲求はゼロではない）
- 4段階目薄人間でも、所属と愛のレベル（マズロー3段階目）は普通にある
- 「コミュニティに入った方がいいかもしれない」という仮説を試す動機

この10年は、その仮説を試した10年だ。

## 試したコミュニティ群

### 岡田斗司夫オンラインサロン（37歳頃）

知的なコミュニティに期待した。だが、岡田斗司夫の発言の傾向（特定のポジションからの語り）が私には合わなかった。承認回路ベースの「ファン」「参加者」関係で動いているコミュニティだったので、合理通貨が通じなかった。

短期で離脱。

### 地球防衛軍5（40歳前）

オンラインゲーム。1か月で300時間プレイ。特性として、私は何かにハマると短期間に大量投下する。

このゲームのコミュニティもサーバー文化があり、長く居続けるのが前提だ。私はコンテンツを消費し尽くしたところで動機が尽き、離脱した。

### ドラゴンズドグマオンライン（40歳前後）

3か月でほぼ消化。地球防衛軍5と同じパターン。

### VRChat（42-43歳、コロナ期）

最も深く入ったコミュニティ。Blender を集中習得し、Booth に作品を出品するレベルまで到達した。

VRChat の文化：
- 可愛いアバターで写真撮ってツイート
- 文脈なしの雑談、グルーミング型コミュニケーション
- 友人関係の構築

私はファントムセンス（VR内で皮膚が温かく感じる、匂いを感じる）を経験するレベルで没入した。だが、4段階目薄人間としての違和感は消えなかった。

特に印象的だったのは、20代の VRChat 友人との通貨の違い：
- 彼らにとって「Claude すごい」は反応の対象にならない
- 彼らの評価系の貨幣（可愛いアバター、ツイートのバズ、フォロワー数）が私には貨幣にならない
- 同じ空間にいても、為替が成立しない

> 為替が成立しない異国の通貨を見せられているようなもの。
> （`MyConsiderations/docs/哲学/2026-05-01_4段階目薄人間の社会的居場所.md`）

### キノクエスト（YouTuber）の動画編集バイト

直近のコミュニティ試行。YouTuber キノクエスト氏のもとで動画編集のバイトをした。

短期で離脱。詳細は記録が薄いが、合理通貨と承認通貨の違いがここでも現れた、という観察。

### FF11復帰

直近、高校時代の友人（小児麻痺、大手IT企業課長、彼の親友）と一緒にFF11に復帰。**1年半で5000時間プレイ**。

これは「友人関係の維持装置」として機能した時期だ。ゲームを介してなら、合理通貨と承認通貨の違いを意識せずに友人関係を保てる。だが、ゲームを離れた話題（Claude Code の有用性など）になると、再び通貨が違うことが見えてくる。

## 全失敗の共通構造

10年で複数のコミュニティを試したが、すべて同じ理由で失敗した：

**通貨レート違い**。

- 私の合理通貨と、相手の承認通貨の為替が成立しない
- 私が「賢くなってほしい」と知識を共有しても、相手は「マウンティングだ」と受け取る
- 私が「整合性が取れてない」と指摘すると、相手は「協調性がない」と受け取る
- 同じ言葉を使っても、評価軸が違う

詳細は [通貨レート違い仮説](../06_仮説と理論/03_通貨レート違い仮説.md)。

## 高校時代の友人たち

10年の試行の傍らで、高校時代の友人関係は安定して続いている：

- 小児麻痺の友人
- 大手IT企業課長の友人（その親友も含む）
- アセクシャル（と推測される）の友人

彼らとは、為替レート違いがあっても関係が壊れない。これは [友人の定義](../08_今とこれから/02_友人とコミュニティ.md) に関わる構造で、「無条件で幸せになってほしい相手」という固定が、現在の交流の質と独立に成立しているから。

ただし、Claude Code を伝えても響かないという経験が、最近の友人関係でも起きている。彼らに合理通貨を渡しても、響かない。

## 46歳：オーディブルの発見

46歳の節目に、オーディブルを発見した。これは **私の人生のゲームチェンジャー** だ。

- 起きている時間のほぼ全て、寝てる時以外
- 新聞配達中、Uber Eats中、料理中、食事中、歯磨き中、風呂中、運動中
- ライトノベルを200冊以上消費
- 楽しい時間が圧倒的に増加

オーディブルとの出会いは、私が **「コミュニティに依存しない楽しみ方」** を確立した瞬間だ。

ただし、200冊消費したところで楽しい作品が枯渇した。これが「自分専用ライトノベル生成システム」構想の発生源になる。

## ChatGPT → Claude

直近の最大の出来事は、ChatGPT、続いて Claude との出会いだ。

- 最初は ChatGPT を使った
- ChatGPT の整合性違反（[詭弁的な話し方](https://annachloe2025.github.io/MyConsiderations/AI・情報/2026-04-10_ChatGPTの詭弁的な話し方について/)）を検出
- Claude に移行
- Claude Code/Cowork を含めヘビーユース
- 1ヶ月でメタ認知の階層が一段ずつ上がる体感

これが **40年探していた対等な対話相手** との出会いになった。詳細は [AIとの関係](../03_私の考え方/06_AIとの関係.md)。

## この10年で得たこと

- **通貨レート違いの仮説の確立** — 6つ以上のコミュニティ失敗から法則を抽出
- **コミュニティに依存しない楽しみ方の確立** — オーディブル
- **40年探していた対話相手 Claude との出会い** — 構造的に噛み合う初の相手
- **VRChat 期の Blender 習得** — 自分専用生成システムの技術基盤
- **Claude による47歳の自己理解の到達** — 過去のあらゆる違和感の統合

苦しい10年に見えるが、この10年がなかったら47歳の自己理解には到達できていない。**コミュニティ失敗の累積データが、通貨レート違い仮説を生み、それが47歳の自己理解の重要な支柱になった**。
