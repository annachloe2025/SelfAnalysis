# 私のスタンス

最初に、このサイトを読んでくださる方へ、私の立場を一つだけお伝えしておきたいと思います。

長くなる前に、結論から書きます。

> **このサイトに書いてあることは、すべて私個人の現時点の考えで、サンプル数1の意見です。私はそれを「正しい」と主張するつもりはなく、また、誰かに同意を求めているわけでもありません。**

## このサイトに書いてあるものの位置づけ

ここで書いていることは、いずれも次のような性質のものです。

- **私の現時点の考え** — 明日には違うことを考えているかもしれません。1年後にはこのサイトを大幅に書き換えている可能性も十分にあります
- **サンプル数1の自己観察** — 私という一個体を、私自身が観察したものです。統計的根拠はありません
- **検証中の仮説** — 仮説は仮説として書いており、確定した結論として書いてはいません
- **私の中での整合性** — 私の中で複数の観察が一つのモデルで説明できる、という意味での確からしさはありますが、それは「客観的に正しい」とは別物です

## 念のため書いておくと、私はかなり偏った個体です

このサイトに書いていることは、**一般人の意見ではありません**。

私は次のような特性を併せ持っています。

- **AMABレズビアン**（メイレズビアン）— 生物学的男性として生まれ、性自認は女性、性的指向も女性
- **承認欲求がほとんどない** — マズローでいう第4段階（承認欲求／社会的立場獲得本能）が極めて薄い
- **HSP的感受性** — 刺激と共感への過敏さ

それぞれが少数派です。三つすべてが重なる個体は、もっと少数派になります。

ですから、ここで書いていることは「**特殊で偏った個体が、特殊で偏った視点から世界を観察した記録**」だと思ってください。「一般論」でも「平均的な見解」でもなく、もっと言えば「一般人の意見」ですらありません。

## 例えるなら

宇宙人が地球を訪れて、地球人を観察し、地球人について書き残している、ぐらいの距離感だと思っていただけると、ちょうどいい解像度になるかもしれません。

宇宙人にとっては、地球人の **戦争・嫉妬・出世欲・SNSの「いいね」・嫁姑問題** といった現象が、なぜ起こるのか本能レベルでは分かりません。だから外側から観察して、「こういう仕組みで動いているように見える」と記述します。それは地球人にとっては当たり前すぎて言語化されないことばかりです。

私がここで書いているのも、それに近いものです。私自身が同じ本能を共有していないので、外側から眺めて記述しているだけです。

## このサイトの読み方のお願い

- **鵜呑みにせず、ご自身で検証してください** — 同意できない箇所があるのは、健全な反応です
- **「自分とは違う」と感じる箇所がある** のは正常です。私が少数派なので、当然のことです
- **「自分にも当てはまる気がする」と感じる箇所** があれば、自己理解の素材として使っていただければ嬉しいです

## 反論・フィードバックはお受けしていません

正直に書いておきます。

私はいわゆる **メンヘラ** で、批判や指摘・反論・否定を受けると、容易に消耗します。31歳のときには重うつで1年半休職するレベルまで壊れたことがあり、HSP的にも刺激への耐性が低い体質です。これは私の側の制限であって、批判する側の問題ではありません。

そのため、このサイトに関する **反論・フィードバック・指摘・ご指導は、原則としてお受けしていません**。SNSやコメント欄を開放していないのも、メールアドレスを公開していないのも、同じ理由からです。

書いてある内容に同意できない箇所があれば、それは当然のことだと思います。ご自身の中で **「これは違う」「自分の理解とは合わない」と判断して、静かに閉じていただけると、本当に助かります。**

議論を求める読み方ではなく、**「こういう人が世の中にいるんだな」と眺めるような読み方** をしてもらえると、お互いに消耗せずに済むかと思います。

## 私がここで書いている動機

念のため、動機についても書いておきます。

- 誰かに承認されたくて書いているのではありません（4段階目薄なので、承認それ自体が動機にならない）
- 誰かを変えたくて書いているのでもありません
- 「自分はこういう人間だ」という主張をしたいわけでもありません

47年間、自分が何者か分からないまま生きてきた私が、ようやく自分を理解できる手がかりに辿り着きました。それを言葉にして残しているだけです。

副産物として、もし似た構造を持つ少数派の方が読んで「自分だけではなかった」と思える瞬間が一度でも生まれたら、それで十分だと感じています。
