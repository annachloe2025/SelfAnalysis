# 承認欲求とは

このサイトで頻繁に登場する **承認欲求** という言葉について、最初に簡単に説明しておきます。

## マズローの欲求段階説

承認欲求は、心理学者アブラハム・マズローが提唱した **欲求段階説（Hierarchy of Needs）** の中の一段階として位置づけられます。

| 段階 | 欲求 | 内容 |
| --- | --- | --- |
| 1 | 生理的欲求 | 食事・睡眠・呼吸など、生命維持の基本 |
| 2 | 安全欲求 | 身の安全、経済的安定、健康 |
| 3 | 社会的欲求（所属と愛） | 家族・友人・恋人・集団に属したい |
| 4 | **承認欲求** | **他者から認められたい、自分を肯定したい** |
| 5 | 自己実現欲求 | 自分らしく生きたい、能力を発揮したい |

第4段階が、このサイトで何度も出てくる **承認欲求** です。

## 私が「承認欲求」と言うときに指しているもの

ここで重要な切り分けをしておきます。

一般的にマズローの第4段階は **他者承認**（他者から認められたい）と **自己承認**（自分自身を肯定したい）の両方を含むものとされます。

ただし、私がこのサイトで「承認欲求」と言うとき、私が指しているのは主に **他者承認** のほうです。さらに私の理解では、他者承認はそれ単体で独立した欲求というより、**もっと根の本能の一形態として発露しているもの** だと感じています。

私の見立てを言葉にすると：

> 人間には、社会の中でポジションを取って、それを自分にとって有利に働かせようとする傾向がある。これは生存と繁殖に有利だった進化的な本能の一部で、その発露の一つの形が「他者から承認されたい」という欲求として表れている。

つまり、**「社会的立場を獲得したい本能」の発露の一形態として、他者承認欲求が機能している** という見方です。

私が「私には承認欲求がほとんどない／薄い」と書くとき、それはこの **「社会的立場獲得本能の発露としての他者承認欲求」がほぼ作動していない** という意味です。出世・肩書・SNSの注目・集団内のポジション争いといった具体的な場面で、私の中で動機がまったく湧いてこない、という感覚に対応しています。

## 自己承認については扱いを保留する

マズロー第4段階のもう一方の側面である **自己承認**（自分自身を肯定したい、自尊心を保ちたい、自分の選択を正当化したい等）については、私はこのサイトでは **扱いを保留** しています。

理由は単純で、私の感覚では、自己承認は他者承認（=社会的立場獲得本能の発露）とは **別系統の欲求** のように感じられるからです。マズローの体系の中で第4段階に一括りにされていますが、私の中では他者承認とは違う回路で動いている気がします。

ただし、これを正確に位置づけるだけの確証は私にはありません。自己承認が実際にはマズローのどの段階に属するべきか、あるいは独立した別の軸として扱うべきかは、私には判断できません。

そのため、当面は **自己承認については議論から除外して考える** という扱いにしています。

このサイトで「私には承認欲求がない」と書く場合、それは原則として **他者承認（社会的立場獲得本能の発露）の不在** を指しており、自己承認については別途の話として保留している、と理解してください。

## 他者承認に絞った具体的な動機

私が「ない」と言っている他者承認の中身を、より具体的に分解すると、次のような動機が含まれます：

1. 評価や賞賛を求める欲求
2. 地位や立場に関する欲求
3. 肩書きや称号への欲求
4. 他者との比較や優越
5. 注目や可視性を求める欲求
6. 集団内評価への欲求

普通の人にはこれらが多かれ少なかれあります。出世したい、褒められたい、SNSで「いいね」がほしい――こういった日常的な動機の多くが、社会的立場獲得本能の発露としての他者承認です。

私の場合、この6項目を一つずつ自己照合した結果、すべてに対して「ほぼ存在しない／薄い／欠けている」という結論になりました（[承認欲求10項目検証](05_根拠とエピソード/05_承認欲求10項目検証.md) 参照、検証ページでは便宜上10項目で検証していますが、本質的には他者承認6項目の不在の話です）。

## 「承認欲求がない」ということの実際

以上の整理の上で、私が「承認欲求がない／薄い」と感じる具体例：

- 出世したい・偉くなりたい欲求が湧かない
- 肩書や称号への憧れがない
- SNSの「いいね」を「気持ち悪い」と感じる
- 集団内のポジション争いに動機が湧かない
- マウンティングをする動機がない

代わりに動いているのは、**合理性と整合性で世界を判定するモード** です。

## 誤解されやすい区別

「承認欲求がない」と言うと、よく次のように誤解されます。これは別物なので、念のため切り分けておきます。

| 混同されがちなもの | 私の場合 |
| --- | --- |
| 「能力を伸ばしたい」（成長欲求） | **強くある**（マズロー5段階目の自己実現に近い） |
| 「知的好奇心を満たしたい」 | **強くある** |
| 「自尊心」「自己肯定感」 | 低い／弱いが、自己承認の話なので扱いは保留 |
| 「友人がいてほしい」（社会的欲求） | 弱くは **ある**（マズロー3段階目） |

承認欲求がない人が、無気力・無関心・無動機なわけではありません。**動機の系統が違う** だけです。

## 承認欲求とは別にある欲求

念のため、もう少し広く付け加えておきます。私が「承認欲求がほぼない」と言っても、人間として動機の回路がスカスカなわけではありません。むしろ次のような欲求は普通に機能しています：

- **性欲** — マズロー1段階目の生理的欲求のレベルで普通にあります（向き先がメイレズビアン的に独特なだけ）
- **ドーパミン系の短期報酬への反応** — ゲーム・面白いコンテンツ・刺激物へのハマり方は、むしろ強いほうです。地球防衛軍5を1か月で300時間プレイしたり、オーディブルでライトノベルを200冊以上消費したりするのは、この回路が普通に動いている証拠です
- **ギャンブル的な「ハマる」感覚** — 脳の報酬回路は普通に機能しています。過去にパチンコや煙草に手を出した時期があり、ハマる感覚自体は私の中にあります（その後、合理性で計算して手放した、という流れ）

つまり、私の中で **生理的欲求・ドーパミン系の刺激追求・脳の報酬回路** は普通に動いているけれど、**社会的立場獲得本能（=他者承認）だけが薄い**、という構造になっています。

「承認欲求がない＝何の欲求もない・無感動な人」ではない、と思ってもらえれば。
