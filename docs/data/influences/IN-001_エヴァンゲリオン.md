# IN-001 エヴァンゲリオン

## 影響源について

新世紀エヴァンゲリオン（1995-1996放映、その後劇場版・新劇場版へ）。
本人は高校終わり頃（1995-1996年頃、18歳前後）にリアルタイムで触れた。

## 本人への影響

- アニメを子供向け媒体としてだけでなく、表現様式として鑑賞する入口になった
- 「制作者のように見る」癖（カット構成・作画の挙動への注意）の形成期と重なる
- 中学から国語克服のため読み始めたライトノベルと併せて、物語没入の習慣を強化した

ただしエヴァ単独の影響を切り出すのは難しく、当時の世代的環境全体（ライトノベル『スレイヤーズ』、推理小説（森博嗣・京極夏彦）など）と並行して効いている。
