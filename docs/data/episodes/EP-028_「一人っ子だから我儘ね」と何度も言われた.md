# EP-028 「一人っ子だから我儘ね」と何度も言われた

## 何が起きたか

子供の頃よく大人の女性から兄弟はいるのと聞かれて、一人っ子ですと答えると、では我儘ねと言われることが多かった。これがコンプレックスとなり、極力人に助けを求めなくなり、色々な面で遠慮もするようになった。我儘だと思われたくなかった。しかしそういう感じで生きていると、自分より他の人達の方が我儘だと感じることが多かった。我儘だと言ってきた大人の女性でさえ自分より我儘なのではないかと思うようになった。30代でうつ病になるが、その原因の一端がこの考え方な気がする。自分でもわかるぐらい自身を抑圧していた。

## 時期

子供の頃

## 後から見るとどう解釈できるか

「私は承認欲求がないので、普通の人より欲求が少ない。だから、叶えたい欲求がそもそも少ないのだ。だから我儘になりにくい体質なのである。その状態で、自身は我儘であるから、それらを律して生きなければいけないと思って修正した結果、過度に他人に頼らなくなったり、欲しいものを口にしなくなったと思う。30代ぐらいでうつ病になるが、その原因の一端がこの考え方な気がする。自身でもわかるぐらい自身を抑圧していた。」
