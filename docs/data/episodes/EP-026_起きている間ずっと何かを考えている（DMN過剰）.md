# EP-026 起きている間ずっと何かを考えている（DMN過剰）

## 何が起きたか

二十歳を過ぎた頃から、起きている間はずっと何かを考えている感覚になった。無意識に思考が続くこの状態はDMN（デフォルトモードネットワーク）と呼ばれるらしい。歩いている時も、車を運転している時も、風呂に入っている時も、常に頭の中で何かを考えていた。物事の道理や本質は何なのかを考え続けていたし、悩みがある時は同じ考えが延々とループすることもあった。この思考の癖が、三十代でうつ病になった一因であることは間違いないと思う。便秘症になってしまったのも、この過剰な思考と無関係ではない気がしている。

## 時期

二十歳過ぎ〜

## 後から見るとどう解釈できるか

「私はHSPの傾向があり、物事を深く考えやすい体質だと言われる。詳しいメカニズムは分からないが、感受性が敏感なため、普通の人より多くのことが気になり、それが思考を促していたのではないかと思う。」「承認欲求がほとんどなかったことも大きい。子どもの頃から「生きている意味とは何だろう」と考えることが多く、生きるための動機が周囲の人より少なかった。」
