# EP-005 銭湯・温泉に強い違和感がある

## 何が起きたか

銭湯や温泉に強い違和感を持っている。男性であっても、他人に裸を見られることに強い恥ずかしさを感じ、複数人がいる空間で裸のまま体を洗う状況では終始落ち着かない。多くの人は銭湯や温泉を「気持ちいいもの」だと感じているようだが、自分には周囲を気にせずにいられるほどの余裕を持ったことはない。

## 時期

大人になっても継続

## 後から見るとどう解釈できるか

「これらの感覚も、今振り返ると、単なる恥ずかしがりや性格の問題だけでは説明しきれないように思える。自分の身体を「見られるもの」「触れられるもの」として強く意識し、それに対して過敏に反応していた点は、これまでに挙げた他の体験とも共通している。」
