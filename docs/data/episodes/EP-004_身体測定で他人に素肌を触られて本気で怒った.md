# EP-004 身体測定で他人に素肌を触られて本気で怒った

## 何が起きたか

身体測定の時間、皆で上半身裸になり体重を測ることがあった。その際、ふざけて冷たい手で他人の体を触り、驚かせて遊ぶことが流行っていた。私は他人に素肌を触られることに強い不快感を覚え、冗談や遊びのつもりで触られても我慢できず、本気で怒っていた。これをきっかけに、上半身裸になる場面ではできるだけ触られないように振る舞ったり、「本当に嫌だから触らないでほしい」とはっきり伝えるようになった。

## 時期

子供の頃（学校の身体測定）

## 後から見るとどう解釈できるか

「自分がメイレズビアンであると認識するようになってから、この記憶の意味が、少し違って見えるようになった。仮に自分が女性であり、望んでいない男性から素肌を触られたとしたら、多くの場合、それは強い不快感を伴うだろう。私の場合、性的指向は女性であり、当時触れてきた相手は、自分にとって対象外の性だった。その状況を当てはめて考えると、あの強い拒否感も、無理のない反応だったのではないかと思えてくる。」
