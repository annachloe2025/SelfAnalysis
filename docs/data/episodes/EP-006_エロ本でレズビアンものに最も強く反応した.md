# EP-006 エロ本でレズビアンものに最も強く反応した

## 何が起きたか

高校生の頃、性処理のためエロ本を使っていた。読んでいるうちに、自分が好きなのはレズビアンものだとわかった。さらに女性同士が対戦してイカセ合うようなものはより好きだった。男性との絡みがあるものは次第に見なくなった。男性が入ってくると興奮しなくなる。それ以来現在に至るまで媒体は本→ビデオ→DVD→PCゲーム→アダルトサイトと変わったが、性的趣向は変わらず同じ。色々と考えたり試した結果、おそらく自分は受け側の女性的な感覚が好きなんだという結論に至った。
