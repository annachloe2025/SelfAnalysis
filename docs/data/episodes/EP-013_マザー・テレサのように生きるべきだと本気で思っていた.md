# EP-013 マザー・テレサのように生きるべきだと本気で思っていた

## 何が起きたか

小学生の頃、尊敬する人はマザー・テレサで、「人はみんなマザー・テレサのように生きるべきだ」と本気で思っていた。どうすれば世界が平和になるのか、いじめがなくなるのかをよく考えていた。

私がそう信じていたのは、承認欲求がない自分にとって、そうすれば簡単に世界が戦争やいじめのない世界になる、と思えたからだ。実際にすべてを捨てて他人のために奉仕するのはできなくても、みんながそういう気持ちを持てば、簡単に平和になると思っていた。

しかし現実には、ほとんどの人が承認欲求を持ち、社会的ポジション取りをして、互いに牽制し合うことで戦争やいじめが起きる。だから、みんながマザー・テレサのように生きるのは不可能だ。

私の場合、承認欲求がないので、マザー・テレサのような思想を持つこと自体は難しくないと考えていた。だからこそ、その思想を「世の中の標準のあり方」として想定してしまっていた。
