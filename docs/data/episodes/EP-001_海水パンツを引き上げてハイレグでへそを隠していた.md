# EP-001 海水パンツを引き上げてハイレグでへそを隠していた

## 何が起きたか

スイミングスクールで、いつも海水パンツをへその位置まで強く引き上げ、ハイレグのような状態にしてへそを隠していた。自分でも少し不自然な格好だとは感じていたが、それでもへそが露出することに強い抵抗があった。小学校5年生頃になって、ハイレグ状態のほうが周囲から見て不自然で、むしろ恥ずかしいのだと気づき、徐々に普通の履き方に戻していった。その過程でも強い恥ずかしさを感じていた。

## 時期

小学1年〜小学5年（スイミングスクール時代、小1〜中3通学）

## 後から見るとどう解釈できるか

「現在の視点から振り返ると、この行動は単なる外見上のコンプレックスだけでは説明しきれないようにも思える。メイレズビアンという自己認識を踏まえると、無意識のうちに、自分の身体を「見られる対象」として強く意識し、過度に隠そうとしていた可能性も考えられる。いわゆる性同一性障害は、脳の性と身体の性が一致しない状態だと説明されることが多いが、もしそうだとすれば、当時の私は、本能的に女性的な身体感覚に近い形で、自分の体を守ろうとしていたのかもしれない。」
