# EP-007 ケーキは一番小さいものを最初に食べるのが正解だと思っていた

## 何が起きたか

不均等に分かれた六等分のケーキがあった場合、一番小さいものを最初に食べるのが正解だと思っていた。ところが周囲の人に聞くと、ほとんどの人が「一番大きいケーキを食べる」と言う。「他人と競争してでも先に手に入れたい」と思うようなものを、子供の頃から今に至るまでほとんど持ったことがない。一番小さいケーキを選ぶのは確信的な感覚で、そうすればその後に残る状態が最も良くなる、という理屈だった。なぜそれを絶対的に正しい選択だと感じていたのか、当時の自分自身にも説明できなかった。

## 時期

子供の頃

## 後から見るとどう解釈できるか

「私は承認欲求が弱いため、そもそも競争しようという発想が乏しく、自分が得をすることに対しても無頓着だった。その結果、物事を「個人の取り分」ではなく、「全体としてどうなるか」で判断しやすくなっていたのだと思う。自分がおいしい思いをすることよりも、全体として最も穏当で、誰も不満を持ちにくい状態になること。その結論が、私にとっては直感的に「正義」として感じられていたのだろう。」
