# EP-031 嫉妬の感情をほとんど持ったことがない

## 何が起きたか

小学生の頃から、嫉妬という感情をほとんど持ったことがなかった。20歳くらいの時には、そのことを自覚していた。当時は原因までは分からなかった。嫉妬の感情がないと、嫉妬している相手の姿がひどく醜悪に見えてしまう。理由もなく他人に嫌がらせをしているように見えた。職場などで嫉妬のあまり優秀な人の足を引っ張る人を見ると、純粋な悪意の行為にしか思えなかった。

## 時期

小学生の頃〜（20歳頃に自覚）

## 後から見るとどう解釈できるか

「承認欲求が希薄であったこと、そしてその背景には、性同一性障害の一種であるメイレズビアンという自己認識があり、体と心の不一致を受け入れる過程で、私の中では承認欲求がそもそも発露しなかったのだと感じている。」嫉妬は承認欲求の派生感情であり、承認欲求がないと嫉妬も生まれない。
