# EP-042 子供の頃から社会には「適応していた」のであって「参加していなかった」

## 何が起きたか

子供の頃から社会には「参加している」感覚ではなく、「社会とはそういうものだから合わせていた」という感覚だった。楽しいと思ったことはない。社会は「自然にいる場所」ではなく「攻略すべき対象」だった。本質を教えない先生には敵対心を持っていた。「攻略するなら仕組みを理解させてくれ」という感覚。メイレズビアンの自覚後、「社会に合わせなければいけない」という感覚が完全に消えた。社会から完全に抜け出した感じ。それまで自分に課していた「社会を攻略する」というミッション自体が、自分ではない誰かのために設定された目標だったと気づいた。

## 時期

子供の頃〜47歳
