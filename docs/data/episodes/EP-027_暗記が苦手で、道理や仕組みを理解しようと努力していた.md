# EP-027 暗記が苦手で、道理や仕組みを理解しようと努力していた

## 何が起きたか

小学生の頃から、物事の本質や道理をとらえることが大事だと考えてきた。単なる丸暗記のような学び方にはあまり意味がなく、その物事がなぜそうなっているのか、どのような理屈で成り立っているのかを理解することのほうが重要だと思っていた。もしかすると単純に暗記が苦手だっただけかもしれないが、暗記に頼るくらいなら、道理や仕組みを理解しようと努力していた記憶がある。

## 時期

小学生の頃〜

## 後から見るとどう解釈できるか

DNA検査で「開放性が高い」（HSPと整合）と判明。さらに「承認欲求がほとんどなかったため、子どもの頃から「生きている意味とは何だろう」と考え、自分なりに生きる理由や価値を作り出そうとしていた」。これらが本質志向の思考を生んだ。
