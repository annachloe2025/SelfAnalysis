# EP-011 「うまい？」と聞かれて「おいしいと思います」と答えた

## 何が起きたか

22歳、越谷市の新聞販売店店長をしていた頃、店の隣の居酒屋でよく昼食を取っていた。ある日のランチがカレー。食べ終わった頃、居酒屋のおばさんが「どう？ うまい？」と聞いてきた。私は少し考えてから「おいしいと思います」と答えた。すると「"おいしいと思います"って何だよ。お前にうまいかどうか聞いてるんだから、おいしいか、おいしくないかだろう」と呆れられた。私は無意識のうちに、そのカレーを「自分がどう感じたか」ではなく「一般的なカレーと比べてどうか」という基準で評価していた。「自分の感覚がおかしいのではないか」と初めてはっきり意識した瞬間だった。

## 時期

22歳（越谷市の新聞販売店店長時代）

## 後から見るとどう解釈できるか

「今になって考えると、これはアイデンティティが非常に希薄だったことの表れだったのではないかと思っている。自分がどう感じたのかよりも、「一般的にどう評価されるか」を優先してしまう。まるで、自分という主体がその場にいないかのような返答だった。」性同一性障害ゆえに守るべき自己像が形成されていなかったこと、承認欲求がないため評価軸を外部に置く構造の現れ。
