# EP-032 特定のフェミニスト女性タレントが嫌いでチャンネルを変えた

## 何が起きたか

20代の頃、テレビには女性のフェミニストが頻繁に出演していた。彼女たちはあらゆる話題を女性問題に結びつけ、「男性が女性を差別しているから悪い」という結論へと導いているように映った。それを疑いようのない正論であるかのように、強い怒りを伴って語る話し方が一つのテンプレートのように感じられた。その女性タレントがテレビに映るだけで、チャンネルを変えてしまうほど嫌悪感を抱いていた。

## 時期

20代

## 後から見るとどう解釈できるか

「私は、承認欲求がほとんどなかったため、この話し方に強い違和感を覚えた。私にはそれが、論点を意図的にずらした詭弁のように見え、事実よりも感情を優先して世の中を扇動しようとしているように感じられた。」「私自身がメイレズビアンであり、女性に対して厳しい視線を向けやすい性質を持っていたことも、こうした感情を加速させていたのだろう。」
