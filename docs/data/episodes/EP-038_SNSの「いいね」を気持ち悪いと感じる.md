# EP-038 SNSの「いいね」を気持ち悪いと感じる

## 何が起きたか

SNSという仕組みがまったく理解できない。なぜ人はSNSをやるのか、なぜ「いいね」をたくさんもらいたいと思うのか、その感覚が分からない。以前あるコミュニティで自分の意見を書いたとき、「いいね」だけが大量に返ってきた。気持ち悪いと感じた。私は自分の意見に対する賛否や、異なる視点からの意見を聞きたかったのであって、無言の反応を集めたかったわけではない。

## 時期

大人（継続）

## 後から見るとどう解釈できるか

「もし、それが「集団の中で認められている」「排除されていない」という感覚を得るための承認欲求なのだとしたら、少なくとも私には、その欲求はほとんど存在しないと断言できる。」自分は意見への賛否や別視点が欲しかったのに、無言の反応（いいね）だけが返ってくることに気持ち悪さを感じた構造。
