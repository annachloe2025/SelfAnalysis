# EP-015 「会話は有益であるべきだ」という意識を持つようになった

## 何が起きたか

高校を卒業したあたりから、「会話は有益であるべきだ」という意識を持つようになった。自分と話すことで、相手が少しでも成長したり、何かを得たりできたらいい、そんな思いで人と接していた。

## 時期

高校卒業以降

## 後から見るとどう解釈できるか

「周囲の人が成長し、有能になっていくほうが全体として望ましいと考えているため、会話の中で知識を共有したり、「こういう考え方がある」「こういう枠組みで見ると分かりやすい」と伝えようとすることがあった。」承認欲求がないので、会話を「自分を見せる場」ではなく「全体最適のための情報共有」として捉える構造。
