# EP-033 知識共有が「頭良く見せたい」と受け取られた

## 何が起きたか

周囲の人が成長し、有能になっていくほうが全体として望ましいと考えていたため、会話の中で知識を共有したり、「こういう考え方がある」「こういう枠組みで見ると分かりやすい」と伝えようとすることがあった。例えば、「マズローの5段階欲求を知っていると人の行動を整理して考えやすい」といった話をすることもあった。しかしこうした行動に対して、「頭が良く見せたいからそんな話をしているのではないか」と受け取られることがあった。次第に他人に対して知識や考え方を説明すること自体を避けるようになり、人と会話をしたいという気持ち自体が薄れていった。現在ではほとんど誰とも会わず、話さない生活をしているが、それで強い不満や欠乏感を覚えることもない。

## 時期

大人（継続）

## 後から見るとどう解釈できるか

「『知識を共有する＝自分を良く見せるため』という発想自体が、私の中にはほとんど存在しないからだ。そのため、私は当初、なぜそのような解釈が生まれるのか理解できなかった。」承認欲求がない自分の動機を、承認欲求がある人がどう翻訳して受け取るかという非対称の典型例。
