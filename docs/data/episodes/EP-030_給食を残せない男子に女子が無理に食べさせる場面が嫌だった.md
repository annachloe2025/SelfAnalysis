# EP-030 給食を残せない男子に女子が無理に食べさせる場面が嫌だった

## 何が起きたか

学校では、いじめられている男子に対して、周囲の女子が「給食を残してはいけない」と言って無理に食べさせようとする場面があった。男子は吐きそうになりながら嫌いな給食を口に運んでいた。それを女子が「残しちゃだめ」と言って食べさせ、先生もそれを容認していた。この状況がものすごく気持ち悪く、強い嫌悪感を覚えた。

## 時期

学校時代（おそらく小学校）

## 後から見るとどう解釈できるか

「私はHSPであるため、いじめられている側に強く共感しやすい。」「私は承認欲求がほとんどないため、この構図そのものが直感的に理解できない。」「もともと女性に対して嫌悪感を抱きやすい体質であり、そのためこの女子の行動が実際以上に冷酷でひどいものとして映ったのだと思う。」3つの特性すべてが重なって嫌悪感を増幅した代表例。
