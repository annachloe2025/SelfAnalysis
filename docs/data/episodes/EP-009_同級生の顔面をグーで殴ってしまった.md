# EP-009 同級生の顔面をグーで殴ってしまった

## 何が起きたか

小学校一年生の頃、教室で「ふざけてぶつかると怪我をすることがあるから危ない」と注意を受けた。同級生のみずの君（仮）が「もののはずみ」という言葉に引っかけて体当たりをして遊ぶのが流行っており、その日の帰り、玄関口で二人きりになったとき、みずの君が「もののはずみ〜」と言って体当たりしてきた。結構な勢いで痛かった。その瞬間、私はみずの君の顔面を思い切りグーで殴ってしまった。みずの君はゆっくりと泣き始めた。私は言葉にできないほど強い後悔と恐怖を感じた。それ以来「意味もなく人を殴ることは絶対にしてはいけない」と強く心に誓った。

## 時期

小学校一年生

## 後から見るとどう解釈できるか

「私は、人を殴ったことで、強烈な罪悪感と恐怖を感じた。」HSP的に他者の痛みを強く感じ取る性質と、暴力を「優位性の確認」として使う動機（承認欲求発露）を持たない構造の表れ。
