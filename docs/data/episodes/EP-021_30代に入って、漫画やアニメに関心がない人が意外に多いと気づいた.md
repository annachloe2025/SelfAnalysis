# EP-021 30代に入って、漫画やアニメに関心がない人が意外に多いと気づいた

## 何が起きたか

30代に入ると、漫画やアニメにあまり関心のない人が意外に多いことに気づいた。多くの人が物語のテーマや世界観を深く理解しようとしていないことにも驚いた。

## 時期

30代

## 後から見るとどう解釈できるか

HSP由来の深い処理レベルで作品を見ている自分と、そこまで深く読んでいない多くの人との「読みの深さの違い」に30代で気づいた。
