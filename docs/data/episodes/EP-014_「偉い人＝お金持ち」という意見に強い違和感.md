# EP-014 「偉い人＝お金持ち」という意見に強い違和感

## 何が起きたか

権力を振りかざすような人は嫌いだった。「偉い人＝お金持ち」という意見にも強い違和感があった。お金や権力を持っているから偉いという考え方は、まったくの間違いだという感覚が強かった。

## 時期

子供の頃

## 後から見るとどう解釈できるか

「権力を振りかざすような人は嫌い」「お金や権力を持っているから偉いという考え方は、まったくの間違いだという感覚が強かった」 — 地位・肩書を価値の源泉にする発想自体が、承認欲求が弱いために自分の中で生まれてこない構造。
