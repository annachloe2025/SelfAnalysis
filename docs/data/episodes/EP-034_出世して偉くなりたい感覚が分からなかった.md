# EP-034 出世して偉くなりたい感覚が分からなかった

## 何が起きたか

20代の頃から「出世して偉くなってお金持ちになりたい」という感覚がよく分からなかった。出世するということは会社の利益や成果に対してより大きな責任を負うこと。例えば五人の部下を持てば、自分を含めた六人分の生産性が上司の判断に左右される。単純に責任は六倍になる。「上に行きたい」「偉くなりたい」という発想自体をほとんど持たずに過ごしてきた。

## 時期

20代〜

## 後から見るとどう解釈できるか

「承認欲求について知ってから、ようやく「なぜ多くの人が出世して偉くなりたいと考えるのか」が理解できるようになった。多くの人にとって、それは能力や責任の問題ではなく、この地位や立場に関する承認欲求を満たす行為なのだと思う。」
