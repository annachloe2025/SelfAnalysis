# EP-040 47歳でメイレズビアンと気づいたとき「自分の中に魂が入った」感覚

## 何が起きたか

47歳で自分がメイレズビアンであると気づいたとき、初めて「これが自分だ」と言える感覚を持つことができた。それは単に説明がついた、というレベルではなかった。自分の中にようやく一つの軸が通ったような感覚があり、「自分の中に魂が入った」と表現したくなるほどの実感が、確かにあった。それまでは、守るべき「自分はこういう人間だ」という像が定まらないまま人生を進んできた感覚があった。

## 時期

47歳

## 後から見るとどう解釈できるか

「アイデンティティを守ろうとする承認欲求が弱かったのではなく、守るべきアイデンティティそのものが長い間形成されていなかった。」47歳でメイレズビアンと自覚したことで、「自分の中に魂が入った」と表現できるほどの軸が生まれた。
