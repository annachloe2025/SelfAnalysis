# EP-002 走り方が「女の子みたい」と何度も言われた

## 何が起きたか

小学生の頃、走り方が女の子みたいだと言われていた。最初に指摘されたのは小学校低学年の頃。何度も言われたのでコンプレックスを感じていた。指摘してきたのはからかいではなく心配してくれている様子の友人だった。当時カメラ付きスマホはなく、自分で確認する手段はなかった。二十歳前後にも同様の指摘を受けたことがある。具体的には、やや内股気味で脚を外側に回すような走り方だったと思われる。自分で「こう走ろう」と意識した記憶はない。

## 時期

小学校低学年〜二十歳前後

## 後から見るとどう解釈できるか

「もちろん科学的な根拠はないが、当時から現在に至るまでの感覚を踏まえると、身体をどう動かすかという無意識の選択において、内的な性自認の影響が表れていた可能性は否定できないように思う。」骨格そのものは男性であるにもかかわらず、無意識に選択される動きが、結果として女性的と受け取られる走り方になっていた、という構造。
