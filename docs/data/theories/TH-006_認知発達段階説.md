# TH-006 認知発達段階説

## 概要

子どもの認知は感覚運動期 → 前操作期 → 具体的操作期 → 形式的操作期と段階的に発達するという理論

提唱者：Jean Piaget（1936年〜）／分野：発達心理学

## 本人がこの理論をどう使っているか

本人の幼少期の特性形成（特にメイレズビアンの自覚以前の身体感覚）を理解する補助枠組み
