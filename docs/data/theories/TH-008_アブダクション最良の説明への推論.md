# TH-008 アブダクション（最良の説明への推論）

## 概要

観察された現象に対して、最もよく説明する仮説を採用する推論様式。演繹・帰納と並ぶ第三の推論

提唱者：C. S. Peirce（1903年〜）／分野：科学哲学

## 本人がこの理論をどう使っているか

本人の自己分析の中核的な推論方法。「もし自分に承認欲求がなかったとすれば、あの出来事はこう説明できる」型の思考
