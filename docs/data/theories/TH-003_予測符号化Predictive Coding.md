# TH-003 予測符号化（Predictive Coding）

## 概要

脳は感覚入力を予測し、予測誤差を最小化することで世界を理解する、という統合的フレームワーク

提唱者：Karl Friston（2005年〜）／分野：神経科学

## 本人がこの理論をどう使っているか
