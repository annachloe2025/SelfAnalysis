# TP-004 慣性期（24-27歳）

## この時期の概要

24歳で経済的自立を達成し、メタ課題が消滅。社交装置が惰性で回っていた3年。

## 主な出来事

- 24歳：経済的自立の達成、上位課題の喪失。実家帰還、アルバイト・日雇い・派遣
- 25-26歳：慣性で日常が回る
- 27歳：慣性が尽きる

## この時期の意味

「駆動源のない行動パターンは3年で崩壊する」という法則を体現した時期。
本来の駆動モード（合理回路）に戻る前の、移行期。
