# TP-002 義務教育期（6-15歳）

## この時期の概要

公立の小学校・中学校に通った 9 年間。水泳・勉強・読書がすべて「素の対象駆動」で動いていた時期。社会への適応がまだ深く要求されない段階。

## 主な出来事

- 1〜13歳まで水泳教室。小4で県大会出場
- 13〜15歳は週5×2時間水泳継続
- 走り方を「女の子みたい」と複数回指摘される（[EP-002](../episodes/EP-002_走り方が「女の子みたい」と何度も言われた.md)）
- ハイレグで海水パンツのへそを隠す（[EP-001](../episodes/EP-001_海水パンツを引き上げてハイレグでへそを隠していた.md)）
- 中学時代に同級生の顔面を殴る、自分も殴られる（[EP-009](../episodes/EP-009_同級生の顔面をグーで殴ってしまった.md), [EP-010](../episodes/EP-010_中学三年生のとき、クラスメートに突然殴られた.md)）
- 中学時代に女性主人公のラノベを無意識に手に取る（[EP-017](../episodes/EP-017_中学生の頃、女性主人公のラノベを無意識に手に取った.md)）

## この時期の意味

社会適応がまだ深く要求されないため、後の本人の駆動モード（合理回路）がそのまま機能していた時期。
本人の生得的特性（メイレズビアン、HSP、承認欲求の不在）がエピソードレベルで現れ始めている。
