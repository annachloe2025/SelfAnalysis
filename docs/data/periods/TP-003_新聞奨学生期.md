# TP-003 新聞奨学生期（15-24歳）

## この時期の概要

15歳から24歳までの9年間。新聞奨学生として上京し、新聞販売店で住み込み勤務をしながら専門学校に通い、最終的に新聞販売店店長まで実装した時期。

経済的自立というメタ課題が、本来薄い社交動機を補い、社会的活動を「課題駆動」として成立させていた9年。9年×2サイクル仮説の前半に該当する。

## 主な出来事（時系列）

- 15歳：高校生アルバイトで経済的自立の初期段階
- 18歳：新聞奨学生として上京、専門学校入学・住み込み勤務開始
- 高校終わり頃〜：エヴァンゲリオンに触れる（[EP-020](../episodes/EP-020_高校終わり頃、エヴァンゲリオンが流行してアニメを見るようになった.md)）
- 専門学校時代：推理小説に触れる（[EP-019](../episodes/EP-019_専門学校時代、推理小説で「頭のいい人はこういう思考をする」と感じた.md)）
- 21〜22歳：新聞販売店店長に昇格
- 24歳：「使われる側でいい」と決断、店長を退き実家へ（[EP-035](../episodes/EP-035_新聞販売店店長を辞めて「一生使われる側でいい」と決めた.md)）

## この時期の意味

「9割側の原理」（=承認回路で動く多数派の世界）を**内側から学習**した期間。これがなければ、後の通貨レート違い仮説や配偶動機-地位獲得本能の連動仮説は、観察基盤を持たない仮説のままだった。
