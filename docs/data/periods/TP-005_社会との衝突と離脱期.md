# TP-005 社会との衝突と離脱期（28-37歳）

## この時期の概要

本来の駆動モード（合理回路）に戻ろうとして、社会と何度も衝突した9年。3度の決定的破綻を経て、社会から離脱する判断に至った。

## 主な出来事（時系列）

- 28歳：軽いうつを発症、転職を繰り返す
- 30歳前後：リーマンショック前後の転職転々
- 31歳：市場勤務（朝3時〜夕方6時、隔週休2日）。**倉庫の裏で本気で泣いた**事件。重うつで1年半傷病手当
- 34-36歳：運送業 3 社連続。いずれも 1 年でキレてやめる（[EP-012](../episodes/EP-012_仕事をしない人を見ると強い憤りを覚え、会社を辞めた.md)）
- 36-37歳：うつ病の時期にアニメで気持ちが楽になる（[EP-022](../episodes/EP-022_20代後半〜30代後半、うつ病の時期にアニメで気持ちが楽になった.md)）
- 37歳：「もう普通の会社員はやらない」決断

## この時期の意味

15-24歳に「9割側の原理」を**内側から学習**した期間と対称的に、本期間は「9割側の原理から離脱する判断」に到達した期間。
本人の自己理解にとって決定的に重要な時期。後の通貨レート違い仮説（[HY-003](../hypotheses/HY-003_通貨レート違い仮説.md)）と配偶動機-地位獲得本能の連動仮説（[HY-001](../hypotheses/HY-001_配偶動機-地位獲得本能の連動仮説.md)）の観察基盤がここで蓄積される。
