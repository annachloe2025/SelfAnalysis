# CL-001 私には承認欲求がほとんどない

## 主張

マズローの欲求段階説における4段階目（承認欲求）が、私においては極めて薄い、あるいはほとんど存在しない。「他者承認」「自己承認」のどちらの方向にも、承認それ自体を求める動機を持っていない。

## 具体的な現れ方

- 出世したい・偉くなりたい欲求がない
- 肩書・称号への憧れがない
- SNSの「いいね」を「気持ち悪い」と感じる
- マウンティングの動機がないので、知識共有が極端にフラット
- 麻雀で勝ちに執着する人を見ると無意識に手を抜いてしまう
- 「使われる側でいい」と決断した時期がある（24歳）

## 検証の経緯

マズローの承認欲求を10項目に分解し、自己照合した結果、すべてに対して「ほぼ存在しない／弱い／欠けている」という結論になった（[マズローの欲求段階説](../theories/TH-001_マズローの欲求段階説.md)）。

具体的な10項目の検証は旧 `自身のエッセイ/04_その考えに至った背景/04-1_/` の各ファイルに本人記述あり。データ層に EP として個別抽出予定。

## 反論への応答

「それは強がりでは」という反論への応答：強がりであれば、承認欲求の不在を認めることに対する**抵抗感**が観察されるはずだが、私の場合、不在を認めることに対する違和感がない。むしろ「ようやく言語化できた」という解放感のほうが強い。

ただしこの反論への応答もサンプル数1の自己観察であり、決定的反論にはなっていない。
