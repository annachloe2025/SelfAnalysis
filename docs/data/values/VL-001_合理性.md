# VL-001 合理性

## この価値観の中身

「合理的であること」を、行動・判断・コミュニケーションの基底として置いている。
具体的には、

- 物事の本質や仕組みが理解できる説明を好む
- 目的に対する手段の整合性を重視する
- 「なぜそれが重要なのか」を問い、答えがないものを優先しない
- 感情的な合意ではなく、根拠のある合意を求める

これは「合理性駆動モード」とも呼ばれる本人の基本動作モードと深く結びついている。

## 行動への現れ

- 出世競争・派閥・空気読みなど、合理性の根拠が薄いものに対して動機が湧かない
- 「みんなで頑張ろう」「期待してるよ」型の動機付けが響かない
- 課題駆動型の業務でモチベーションが最大化する
- ChatGPT が言い方をずらしているとき、整合性違反として高感度に検出する

## 形成の経緯

形成期の特定は困難。本人記述では「気づいたときには既にこういう判定をしていた」とされる。
推定としては、

- 承認欲求が薄いため、社会的合意よりも合理的判定が動作モードとして優位になりやすい
- HSP的感受性により、整合性違反のノイズに敏感に反応する
- 一人っ子・シングルマザー家庭という環境で、独立した判定を要求される場面が多かった可能性

これらは仮説であり、本人ヒアリングで検証要。

## 対立する価値

- **立場・序列を重んじる価値観**：合理性より「誰が言ったか」を優先する判定。本人とは強い摩擦
- **同調圧力**：理由を問わずに合わせることを求める価値観。31歳の倉庫の裏での体験はこれと合理性の衝突に帰される
- **承認回路ベースの動機付け**：本人にとって動かない動機
