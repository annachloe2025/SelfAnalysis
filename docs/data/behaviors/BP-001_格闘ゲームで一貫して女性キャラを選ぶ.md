# BP-001 格闘ゲームで一貫して女性キャラを選ぶ

## 行動の中身

格闘ゲームをプレイするとき、プレイヤーキャラクターとして**女性キャラを必ず選ぶ**。
高校時代から一貫しており、特定のシリーズに限定されない。

## 行動の特徴

- 強弱・性能でキャラを選ぶのではなく、性別が判定基準
- 「強いから」「使いやすいから」という合理的説明では本人の動機を説明しきれない
- 本人にとっては「自然にそうしている」感覚

## この行動から見えるもの

- 性自認女性（メイレズビアン、[CO-001](../concepts/CO-001_メイレズビアン.md)）の手がかり
- 「自分が動かしたい身体」として女性キャラを選んでいる、という解釈
- 当時は性自認の概念を持っていなかったため、行動の意味を言語化できていなかった
