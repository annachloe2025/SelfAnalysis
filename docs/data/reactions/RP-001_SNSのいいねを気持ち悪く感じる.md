# RP-001 SNSのいいねを気持ち悪く感じる

## トリガー

- 自分の投稿に「いいね」が付くのを見たとき
- 他人の投稿で「いいね」を求める振る舞いを見たとき
- 「いいね」を集めることを目的化した投稿全般

## 反応の中身

- 「気持ち悪い」という直接的な嫌悪感
- 賛否や異なる視点からの意見が欲しいのに、無言の反応が集まる構造への違和感
- 機能を切る、SNS自体から距離を取る

本人の言葉：

> 気持ち悪いと感じた。私は自分の意見に対する賛否や、異なる視点からの意見を聞きたかったのであって、無言の反応を集めたかったわけではない。
> （`自身のエッセイ/04-1-8_集団の中での評価に関する欲求について.md` より）

## 反応の起源

- 「いいね」機能が承認回路を駆動する設計であることを、本人が直感的に検出している
- 本人は承認回路で動いていない（[CL-001](../claims/CL-001_私には承認欲求がほとんどない.md)）ため、「いいね」がノイズに感じられる
- 価値観としての合理性（[VL-001](../values/VL-001_合理性.md)）の観点では、「いいね」は意見表明として情報量が少なく、合理的判定の材料にならない
