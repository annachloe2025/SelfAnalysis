# CO-008 戦略的撤退

## 短い定義

世界平和や社会全体の改善は構造的に実現不能と判定し、自分の楽しさを目的とする生活へ撤退する判断

## 詳細な定義

37歳で「もう普通の会社員はやらない」決断、46歳のオーディブル発見、47歳の自己理解への到達などを統合する判断軸。諦めではなく、構造的判断としての撤退。

## なぜこの言葉が必要か

既存の一般語では、本人が記述したい現象を正確に指せないため。
