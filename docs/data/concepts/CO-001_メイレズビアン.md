# CO-001 メイレズビアン

## 短い定義

生物学的男性、性自認女性、性的指向女性の三条件を満たす人。

## 詳細な定義

次の三条件をすべて満たす：

1. **生物学的性**：男性（出生時に割り当てられた性別が男性）
2. **性自認**：女性
3. **性的指向**：女性

「メイレズビアン」は本人が自己認識のために使う用語であり、医学的・公式な用語ではない。AMABレズビアン（Assigned Male At Birth レズビアン）、male lesbian も同義語として使う。

## なぜこの言葉が必要か

既存の用語では言い表せないため。

- 「ゲイ男性」：性自認が男性で性的指向が男性、なので異なる
- 「トランス女性（MtF）」：性自認が女性で性的指向が**男性**の場合、なので異なる
- 「トランス女性レズビアン」：医学的な手術・移行を経た人を指す印象が強く、本人の状況（移行手術なし）と一致しない

本人の場合、移行手術や精神科診断を経ておらず、「気づいたとき47歳」という状況も含めて、独自の用語が必要だった。

## 由来・初出

47歳の誕生日後ほぼ1ヶ月で、メイレズビアンとしての自己認識に至った。
初出は本人エッセイ `01_性同一性障害メイレズビアン/01-1_/01-1-0_メイレズビアンとは.md`。

## 類似語との違い

| 用語 | 違い |
| --- | --- |
| トランスジェンダー全般 | より広範な用語。メイレズビアンは性自認女性 + 性的指向女性に特定 |
| FtM レズビアン | 出生時女性なので生物学的性が異なる |
| トランス女性 | 性的指向が女性とは限らない |
| クロスドレッサー | 性自認の問題ではないので異なる |

## 用例

> 自分が男性でありながら性自認は女性で、性的指向が女性であることに気が付いたのは、47歳の誕生日を過ぎてから一か月ほど経った頃だった。
