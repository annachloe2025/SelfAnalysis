# CO-010 合理性駆動モード

## 短い定義

承認回路ではなく、合理性と整合性で世界を判定して動く本人の基本動作モード

## 詳細な定義

「私は承認欲求がない」の積極的言い換え。本人の動機・判断・コミュニケーションがすべて合理性ベースで動くことを記述する。
