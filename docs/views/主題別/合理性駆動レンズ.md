# 合理性駆動レンズ

承認欲求が薄い本人の場合、判断と動機の中心には**合理性と整合性で世界を判定する**モードが置かれている。これを「合理性駆動モード」と呼ぶ。

これは「優れた知性」ではなく、**承認回路が機能しないため、必然的にこのモードで動かざるを得ない**という構造的な帰結である。

## このビューでの解説

合理性駆動モードは、本人にとって「考え方」というより**動作モードそのもの**である。意識的に選んでいるわけではなく、承認回路が機能しないために必然的にこのモードで動く。

### 動作の具体例

- ChatGPT が言い方をずらしている → 整合性違反として高感度に検出
- 「みんなが言ってるから正しい」型の合意 → 動機にならない
- 「偉い人 = お金持ち」 → 命題として違和感を覚える
- ケーキの分け方 → 全体最適の問題として処理

### 整合性による真理性の感覚

本人の確信は、**独立観察が一つのモデルに収束する**ことで生まれる。
47歳のメイレズビアンの自覚が決定的だったのは、過去の無数の違和感（走り方、ハイレグ、レズビアン嗜好、女性キャラ選択、嫁姑嫌悪、嫉妬の不在、出世欲のなさ、SNS拒否、コミュニティ失敗 etc.）が、この一つの認識でほぼすべて説明されたから。

これは [HY-006](../../data/hypotheses/HY-006_整合性による真理性の感覚.md) として仮説化されている。確証バイアスとの区別は「**反証可能な状態で複数観察を集めたか**」にある（が、サンプル数1の留保は付く）。

### 戦略的撤退

合理性駆動モードで世界を判定すると、世界平和や社会全体の改善は構造的に実現不能と判定される（多数派が承認回路で動く以上、合理通貨では動かせない）。
そのため、本人は「自分の楽しさ」を目的とする生活へ**戦略的に撤退**した（[CO-008 戦略的撤退](../../data/concepts/CO-008_戦略的撤退.md)）。

> 諦めではなく、構造的判断としての撤退。
