# 承認欲求レンズ

このビューは、本人の生得的特性のうち**承認欲求がほとんどない構造**に関連する atom を一望する。

マズローの欲求段階説における4段階目（承認欲求）が、本人の場合は極めて薄い、あるいはほとんど存在しない。これは「強い」「優れている」のではなく、**生得的にこの欲求回路が機能していないがゆえに**、特定の動機付けが効かないという構造である。

## このビューでの解説

承認欲求の不在は、本人の人生のあらゆる場面で観察される。
重要なのは、**特定の項目だけが薄いのではなく、マズロー4段階目に列挙される10項目すべてに対して系統的に薄い**という点である。本人はこれを一つひとつ自己照合し、「ほぼ存在しない／弱い／欠けている」という結論を得た。

### 代わりに動いているもの

承認欲求の代わりに動いているのは、**合理性駆動モード**（[CO-010](../../data/concepts/CO-010_合理性駆動モード.md)）である。
判断・行動・コミュニケーションのすべてが、合理性と整合性を基準として組み立てられる。これは [合理性駆動レンズ](合理性駆動レンズ.md) で扱う。

### 承認欲求の不在がもたらす帰結

- 派閥・空気読み・ポジション取りで動かない → 組織内で「変わった人」と認識される
- マウンティング動機がない → 知識共有が極端にフラット → ときに「頭良く見せたい」と誤解される（[EP-033](../../data/episodes/EP-033_知識共有が「頭良く見せたい」と受け取られた.md)）
- 「みんなで頑張ろう」「期待してるよ」が動機にならない → 課題駆動型の業務でしか機能しない
- 47年間アイデンティティが不明確 → 「守るべき自分」が形成されにくい

### 通貨レート違いとの関係

合理通貨で動く本人と、承認通貨で動く多数派の間で**為替が成立しない**現象が、本人の人生における 6 つのコミュニティ失敗を構造的に説明する（[HY-003](../../data/hypotheses/HY-003_通貨レート違い仮説.md)）。
