# HSPレンズ

このビューは、本人の生得的特性のうち**HSP的感受性**（Highly Sensitive Person）に関連する atom を一望する。

HSP は刺激への過敏さと共感力の強さを特徴とする生得的気質であり、本人の場合、これがアニメ・小説への深い没入と、社会の不合理への強い反応の両方を生んでいる。

「強み」ではなく**生得的にこの感受性を持って生まれたがゆえに**、特定の環境では消耗が累積する体質である。

## このビューでの解説

HSP は単独の特性として観察されるよりも、本人の場合、**承認欲求の不在 × メイレズビアン**との組み合わせで効果が増幅する。

- HSP × 承認欲求の不在 → 世間の評価に依存しない独自の物語選択。「みんなが楽しんでいるもの」より「自分のツボに刺さるもの」を貫ける
- HSP × メイレズビアン → 物語の女性視点に身体ごと持っていかれる過敏さ
- HSP × 合理性駆動モード → 整合性違反の検出が高感度になる（ChatGPT が言い方をずらしているのに気づくレベル）

### HSP のネガティブ側面

本人の場合、対面コミュニケーションでの消耗、感情レイヤー会話の疲労、長時間労働での精神的崩壊（31歳の市場勤務、34-36歳の運送業）など、HSP がもたらす負荷も大きい。
これは [TP-005 社会との衝突と離脱期](../../data/periods/TP-005_社会との衝突と離脱期.md) と密接に結びつく。

### 検証密度の現状

旧 SelfAnalysis では、HSP の章が承認欲求と比べて検証密度が低いと指摘されていた（[エッセイ分析レポート](../../../エッセイ分析レポート.md)）。
今後の Phase 7 で、マズロー10項目検証と同じ粒度で HSP の特性を体系的に検証する作業が予定されている。
