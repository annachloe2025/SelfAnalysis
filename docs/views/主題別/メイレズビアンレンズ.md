# メイレズビアンレンズ

このビューは、本人の生得的特性のうち**メイレズビアン**（生物学的男性、性自認女性、性的指向女性）に関連する atom を一望するための入口である。

> 47歳の誕生日後ほぼ1ヶ月で、自分がメイレズビアンであることに気づいた。
> それまでの47年間、自分が他人と何が違うのかを探し続けていた違和感が、この一つの認識で氷解した。

これは「強み」「特技」ではなく、**生得的にこの体質を持って生まれたがゆえに、現状こうなってしまった**という、構造的な出発点として読まれることを想定している。

## このビューでの解説

メイレズビアンとしての自己認識は、47歳という遅い時期に成立した。それまで、本人にはこの言葉も概念もなく、無数の違和感だけが累積していた状態だった。

メイレズビアンであることが、本人の生涯にもたらした帰結は次のように整理できる。

1. **身体感覚レベル**：男性身体への違和感（へそ、銭湯、身体測定）と、無意識の女性的所作（走り方）。本人は意識的に選択していないが、結果として周囲から指摘される
2. **性的嗜好レベル**：レズビアン表現への一貫した嗜好、女性キャラへの感情移入。「自分が動かしたい身体」「自分が同一化したい視点」が無意識に選ばれる
3. **アイデンティティ・レベル**：47年間アイデンティティが不明確だった。これは性自認と身体の不一致が、自我形成の基底を不安定にしたためと推定される
4. **承認欲求との連動**（仮説）：配偶ゴールが現実空間で成立しない構造が、地位獲得本能の発動を抑え、結果として承認欲求の薄さを生んだ可能性（[HY-001](../../data/hypotheses/HY-001_配偶動機-地位獲得本能の連動仮説.md)）

### 女性への複雑な感情について

メイレズビアンであることと「実在の女性への嫌悪」が共存する点は、しばしば誤解を招く。
本人の認識では：

- 抽象的な「女性性」「女性視点」への関心は強い（性自認女性として）
- 実在の女性に対しては、男性視点での観察が含まれることがあり、嫌悪感が生じる場面がある
- これは投影バイアス（自分の価値観を他者に投影してしまう）の認知の歪みである可能性

詳細は [CL-011 私は実在の女性に対して複雑な感情を抱く](../../data/claims/CL-011_私は実在の女性に対して複雑な感情を抱く.md) を参照。
