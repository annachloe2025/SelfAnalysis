# AI引き継ぎビュー

このビューは、Claude / ChatGPT などの AI を継続的な対話相手として運用する前提で、**最初に読ませる人物像**として設計されている。
機械可読性と必要十分な人物把握を両立する。

## 30秒で把握する hoehoe

- **生年**：1977年（執筆時点 47-49歳）
- **性別**：出生時 男性／性自認 女性／性的指向 女性 → **メイレズビアン**（[CO-001](../../data/concepts/CO-001_メイレズビアン.md)）
- **三つの生得的特性**：
  1. メイレズビアン（性同一性）
  2. 承認欲求がほとんどない（マズロー4段階目が薄い）
  3. HSP的感受性
- **47歳でこれらが繋がり、47年間の違和感が氷解した**
- **動作モード**：合理性駆動（[CO-010](../../data/concepts/CO-010_合理性駆動モード.md)）。承認回路ではなく合理性と整合性で世界を判定
- **戦略的撤退**（[CO-008](../../data/concepts/CO-008_戦略的撤退.md)）後、自分の楽しさを目的とする生活に移行
- **創作プロジェクト**：自分専用ライトノベル生成システム（AI支援）

## トーン規約（必読）

このフォルダで本人について書くとき、AI は次を守ること（詳細は [CLAUDE.md §8](../../../CLAUDE.md)）：

- 一人称は **「私」**
- 文体は「である」調を優先（既存の「です・ます」混在は許容）
- **「サンプル数1」「あくまで仮説」「現時点での結論」** という相対化のフレーズを大事にする
- **断定を避ける慎重さ** と **本人の発見への確信** を両立
- 機微情報（性同一性、うつ病歴、職歴の挫折）は隠さず正確に書く

### 生得的特性の framing（重要）

「強み」「特技」「武器」「ユニークな才能」と書かない。
「**生得的にこの体質を持って生まれたがゆえに、現状こうなってしまった**」という構造的・記述的な framing で書く。

- ❌ NG: 「三本柱」「私の強み」「ユニークな才能」「〜を活かして」
- ✅ OK: 「三つの生得的特性」「先天的にこの構造を持つ」「〜という構造ゆえに」
