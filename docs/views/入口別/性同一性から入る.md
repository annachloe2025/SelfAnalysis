# 性同一性から入る

このページは、性同一性をきっかけにこのサイトに来た読者向けの導線。

---

## 30秒で：私は誰か

47歳になって、自分が**メイレズビアン**（生物学的男性、性自認女性、性的指向女性）であることに気づいた。
それまで47年間、自分が他人と何が違うのかを探していた違和感が、この一つの認識で氷解した。

> 自分が男性でありながら性自認は女性で、性的指向が女性であることに気が付いたのは、47歳の誕生日を過ぎてから一か月ほど経った頃だった。
> 過去に起きていた不可解な出来事が、一気につながったような感覚があった。
