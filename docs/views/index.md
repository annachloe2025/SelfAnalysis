# ビュー層 — 多面的な「私」の見え方

ビュー層は、データ層の atom を**特定の視点で束ね直した読み物**である。
本文を持たず、データ層への参照と視点別のナラティブで構成される。

## ビューの 7 軸

| 軸 | フォルダ | 用途 |
| --- | --- | --- |
| [主題別](主題別/index.md) | `主題別/` | テーマ・トピックで束ねる（メイレズビアン、HSP、承認欲求、合理性駆動 等） |
| [時系列](時系列/index.md) | `時系列/` | 時間軸で束ねる（9年×2サイクル、簡易年表、ライフフェーズ別） |
| [抽象度別](抽象度別/index.md) | `抽象度別/` | データタイプ別の俯瞰（全エピソード索引、全主張カタログ） |
| [関係別](関係別/index.md) | `関係別/` | 「私と何か」の関係軸（私と社会、私と他者、私と AI 等） |
| [用途別](用途別/index.md) | `用途別/` | 読まれ方の用途（履歴書、初対面用、AI引き継ぎ、創作素材） |
| [信頼度別](信頼度別/index.md) | `信頼度別/` | 確信レベル（確信していること、仮説の段階、まだわからないこと） |
| [入口別](入口別/index.md) | `入口別/` | 初見の読者の興味から入る（性同一性から、思想から、キャリアから） |

詳細は [_design/04_ビュー層カタログ.md](../../_design/04_ビュー層カタログ.md) を参照。
