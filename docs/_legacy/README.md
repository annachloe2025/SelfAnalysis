# docs/_legacy/

このフォルダは **新アーキテクチャ移行中（2026-05-08〜）** に、旧 docs 構造のファイルを一時的に退避させる場所である。

## このフォルダの役割

- 旧構造（8章ベース）のファイルを移行中に保持する
- mkdocs のナビゲーション（mkdocs.yml の `nav`）からは**外す**ため、公開サイトには表示されない
- 移行完了後、`_archive/旧素材_2026-05-08/` へ最終退避するか、必要なら data/ または views/ に統合する

## なぜ docs/ 配下か

`_archive/` は `.gitignore` で除外されているが、`docs/_legacy/` は git で履歴を追跡したい（誰がいつ何を退避したか残す）ため、`docs/` 配下に置く。
ただし mkdocs ビルド時には除外される（mkdocs.yml の `exclude_docs` で指定）。
