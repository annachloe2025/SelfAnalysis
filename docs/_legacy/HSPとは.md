# HSPとは

このサイトで使う **HSP** という言葉について、最初に簡単に説明しておきます。

## HSP の定義

**HSP** は **Highly Sensitive Person**（ひどく敏感な人）の略で、心理学者エレイン・アーロン博士が1996年頃から提唱している概念です。

「病気」や「障害」ではなく、生まれつきの **気質（temperament）** の一つだとされています。人口の15〜20%程度がこの気質を持つと推定されています（神経生物学的には、新奇刺激への深い処理を担う脳回路が他より活性的、と言われます）。

## DOES — HSP の四つの特徴

アーロン博士は、HSP には以下の **4つの特徴** がすべて揃っているとしています。頭文字を取って **DOES**。

| 略号 | 特徴 | 内容 |
| --- | --- | --- |
| **D** | Depth of processing | 情報を **深く** 処理する。表面的に流さず、構造や意味を掘る |
| **O** | Overstimulation | 刺激に **圧倒されやすい**。人混み・騒音・長時間の対面で疲れやすい |
| **E** | Emotional reactivity & Empathy | 感情の **反応が強く、共感も高い**。物語に没入する、不当な扱いを長く引きずる、他者の痛みを自分のことのように感じる |
| **S** | Sensing the subtle | **微細な刺激への感受性**。非言語的手がかり、声のトーン、空気の変化、わずかな違いに気づく |

私はこの4項目すべてに当てはまっている自覚があります。

## よくある誤解との切り分け

HSP は「最近よく聞くけど、何となく曖昧な言葉」として広まっている面があり、いくつかの誤解があります。最初に切り分けておきます。

| 混同されやすいもの | HSP との違い |
| --- | --- |
| **内向的（introvert）** | 別物。HSP の中に外向的な人もいる（推定30%程度） |
| **発達障害（ASD・ADHD等）** | 別物。診断基準もメカニズムも違うが、症状が部分的に似ることはある |
| **メンタルが弱い／神経質** | 「弱さ」ではなく「気質」。情報処理が深いゆえの疲労 |
| **自己診断による流行り言葉** | 学術的には Aron 夫妻による HSPS スケール（Highly Sensitive Person Scale）等で測定可能 |

## HSP の長所と短所

HSP は短所だけでも長所だけでもありません。両側面があります。

**長所側**

- 細部によく気づく
- 共感力が高く、他者の感情を察知する
- 物事を深く考える
- 美的感覚・芸術への感受性が豊か
- 倫理観が強い

**短所側**

- 刺激の多い環境で消耗しやすい
- 他者の感情に巻き込まれやすい
- 批判・否定への過敏な反応
- 長時間労働・対面が続くとメンタルが折れやすい
- 一般的な労働環境への適応の困難

## 私の場合の特殊性

私の場合、HSPに加えて、**メイレズビアン（性同一性）** と **承認欲求の不在** という2つの特性が重なっています。この三重組み合わせがどれぐらいの希少性かは分かりませんが、おそらくかなりの少数派です。
