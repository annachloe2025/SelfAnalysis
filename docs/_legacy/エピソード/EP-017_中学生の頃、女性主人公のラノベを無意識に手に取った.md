# EP-017 中学生の頃、女性主人公のラノベを無意識に手に取った

**時期:** 中学生

**内容:**
中学生の頃、なぜか国語だけが苦手で、改善のため小説を読んでみようと思いライトノベルを読み始めた。書店で自分の感覚だけを頼りに選んでいたが、『スレイヤーズ』や『燃える瞳のメル』など、女性主人公の作品を無意識に手に取ることが多かった。
