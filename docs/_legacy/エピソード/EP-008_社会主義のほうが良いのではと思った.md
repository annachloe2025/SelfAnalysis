# EP-008 社会主義のほうが良いのではと思った

**時期:** 小学四〜五年生（公民の授業）

**内容:**
公民の授業で資本主義と社会主義について学んだとき（ちょうどソ連崩壊の年の前後）、なぜ社会主義よりも資本主義の方が推奨されるのかがまったく理解できなかった。「みんなで協力する社会」の方が良いのではないか、競争で勝ち負けが決まる社会よりは全員で助け合う社会の方が素直で健全に見えた。なぜ社会主義は必然的に独裁国家になってしまうのか、その発生源がまったく見えなかった。長い間この点について腑に落ちない感覚を持ち続けた。

## 関係する特性

**承認欲求の不在**

## 考察（原本より）

「自分には承認欲求がほとんどないのではないか、と気づいてから、この疑問がようやく理解できるようになった。多くの人は、承認欲求に基づいて行動している。承認欲求を前提に社会主義的なシステムを構築しようとすると、評価・立場・影響力を巡る力関係が必然的に生まれ、最終的に権力が集中し、独裁に近い構造になってしまう。その感覚が、承認欲求の弱い私には、長い間まったく想像できなかったのだ。」
