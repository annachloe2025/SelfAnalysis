# EP-037 カラフルな服が好きでない、白か黒、特に黒を好む

**時期:** 大人（継続）

**内容:**
街を見ているとカラフルな服を好んで着る人がいる。私はそういった服装が昔からあまり好きではない。若い頃は自分でもはっきり分かっていなかったが、今振り返ると白か黒の服を好む傾向がある。特に黒が好きだと今ならはっきり言える。極力目立ちたくなく、可能であればひっそりと暮らしていたいと感じている。

## 関係する特性

**承認欲求の不在**

## 考察（原本より）

「目立つと面倒そうだからとか、目立つための努力が大変そうだからやらない、という次元の話ではない。そもそも、「目立とう」「注目されよう」という発想自体が、自分の中にはほとんど浮かんでこない。」服装の好みは、注目欲求がないことの観測可能な現れ。
