# EP-035 新聞販売店店長を辞めて「一生使われる側でいい」と決めた

**時期:** 20代前半

**内容:**
20代前半の頃、都内で新聞販売店の店長をしていた。新聞奨学生（学費を稼ぐために学校に通いながら新聞配達を行う学生たち）を雇用していた。営業活動なども学生に任せることになる。彼らへの負担を増やせば店の成績は上がり、減らせば下がる。仕事をしているうちに、「若者の未来を搾取しているのではないか」という感覚を強く持つようになった。新聞販売店を辞める決断をした。世の中には「使う側」と「使われる側」が存在するという構造を、はっきりと自覚した瞬間だった。そのとき、自分は一生、使われる側でいいと決めた。それ以来、意識的に「使う側」「立場を持つ側」になろうとする道を選ばずに生きてきた。

## 関係する特性

**承認欲求の不在**、**HSP**

## 考察（原本より）

「肩書きや称号を欲する承認欲求は、明確に存在しないと断言できる。」「人生の中で、「使う側として生きるか」「使われる側として生きるか」を意識的に選び、なおかつ後者を選択する人は、そう多くはないと思う。」HSP的に「若者の未来を搾取しているのではないか」と強く感じ取り、承認欲求がないので「使われる側」を選ぶことに迷いがなかった構造。
