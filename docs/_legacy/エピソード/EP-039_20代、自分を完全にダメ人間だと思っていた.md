# EP-039 20代、自分を完全にダメ人間だと思っていた

**時期:** 20代

**内容:**
20代の頃、自分を完全にダメ人間だと思っていた。煙草を吸い、パチスロをし、借金を抱え、仕事をしては一年ほどで辞める、という生活を繰り返していた。煙草をやめよう、パチスロをやめよう、生活を立て直そうと何度も考え、実際に試しては挫折し、そのたびに「自分は本当にダメな人間だ」と思っていた。今になって振り返ると、当時の私は「何もしていなかった」わけではなく、むしろどうにか生き方を見つけようとして考え、努力し、何度も試行錯誤していた。しかしその事実を自分自身で評価することができていなかった。

## 関係する特性

**承認欲求の不在**、**性同一性障害**

## 考察（原本より）

「私は承認欲求が弱かった、あるいは欠けていたために、生きる指針を自然に持つことができなかったのではないかと思っている。多くの人は、承認欲求を満たす方向に進めば、ある程度「これでいいのだ」という感覚を得られる。他人からの評価や、集団の中での位置づけが、そのまま生きる指針になる。しかし私の場合、そうした指針がほとんど働かなかった。」「実際の行動量と、自己評価がまったく噛み合っていなかった」 — 性同一性障害（自分が何者かが分からない）と承認欲求の不在（外部評価軸が機能しない）が重なって自己評価が極端に低くなった。
