# EP-014 「偉い人＝お金持ち」という意見に強い違和感

**時期:** 子供の頃

**内容:**
権力を振りかざすような人は嫌いだった。「偉い人＝お金持ち」という意見にも強い違和感があった。お金や権力を持っているから偉いという考え方は、まったくの間違いだという感覚が強かった。
