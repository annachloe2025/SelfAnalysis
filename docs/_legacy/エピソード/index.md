# エピソード一覧

このセクションは、私が過去に書いたエッセイ・メモから個別の体験・観察・出来事を抽出したエピソード集です。

各エピソードは1ページ1件で独立しており、後で分析しやすいようタグを暫定的に付けています。

## タグの意味

- **時期**: `#子供時代` `#小学校` `#中学校` `#高校` `#20代` `#30代` `#40代`
- **特性軸**: `#性同一性障害` `#承認欲求の不在` `#HSP`
- **ジャンル**: `#身体感覚` `#性嗜好` `#人間関係` `#仕事` `#物語愛` `#思考傾向` `#嫉妬・競争` `#嫌悪感` `#自己評価` `#社会観` `#依存` `#うつ`
- **性質**: `#根拠寄り` / `#結果寄り` / `#両義`
  - **根拠寄り**: その特性があると判断する材料になっている事象
  - **結果寄り**: その特性があるからこそ起きた事象
  - **両義**: 根拠でも結果でもありうる事象
