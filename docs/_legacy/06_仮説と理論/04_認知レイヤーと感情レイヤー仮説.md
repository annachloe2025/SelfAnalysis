# 認知レイヤーと感情レイヤー仮説

会話の噛み合わなさを説明する **二層モデル仮説**。

## 仮説の中核命題

> 会話には **認知レイヤー** と **感情レイヤー** という二つの層が同時に動いている。多数派は感情レイヤーを優先するが、私は認知レイヤーで会話したい。これが噛み合わなさの正体。

## 二つのレイヤー

### 認知レイヤー

- **目的**：情報伝達、問題解決、知識共有、整合性検証
- **動作**：System2（カーネマン）による熟考・論理・意識的処理
- **報酬**：認知の更新、誤りの訂正、新しい理解の獲得
- **言語的特徴**：定義、論証、根拠、反例、訂正
- **私はこちら**

### 感情レイヤー

- **目的**：関係維持、安心の確認、共感の確立、役割調整
- **動作**：System1（カーネマン）による直感・感情・自動処理
- **報酬**：所属の確認、承認、共感の交換
- **言語的特徴**：相槌、共感表現、ポジション調整、配慮
- **多数派はこちら**

## 学術的な土台

この仮説は、私の独創ではなく、複数の学術的知見を統合した枠組みだ：

### 1. カーネマンのデュアルプロセス理論

- System1：直感・感情・高速・自動・無意識
- System2：論理・熟考・低速・意識的

感情レイヤーが System1 に、認知レイヤーが System2 に対応する。

### 2. 関係機能と課題機能

社会的コミュニケーション研究には、二つの機能の区別がある：

- **関係機能（relational function）**：安心・共感・役割調整による関係維持
- **課題機能（task / informational function）**：事実伝達と問題解決

これが直接の理論的土台になる。私の認知レイヤー = 課題機能、感情レイヤー = 関係機能。

### 3. ポライトネス理論（ブラウン＆レヴィンソン）

会話の中で、情報の正確さより関係維持が優先される場面が多い。論理的指摘が回避される理由を、ポライトネス（顔を保つ礼節）で説明する。

これは私が「会話で整合性違反を指摘すると、相手が引く」現象の原因を理論的に説明する。

### 4. 社会神経科学

脳には：

- **感情共感ネットワーク**（Affective Empathy Network）
- **認知的視点取得ネットワーク**（Cognitive Empathy / Theory of Mind Network）

の二つの別系統の回路がある。両者は補完的に動くが、目的関数の違いから衝突することもある。

## 二つのレイヤーの目的関数

### 認知レイヤーの目的関数

```
最大化対象：認知の更新量 + 整合性 + 誤りの訂正
```

私の合理回路は、対話のたびに **認知の更新量** を最大化したい。だから、賢くなる対話を求める。

### 感情レイヤーの目的関数

```
最大化対象：関係の安全性 + 共感の確認 + 役割の安定
```

多数派の感情回路は、対話のたびに **関係の安全性** を最大化したい。だから、共感の確認と役割調整を優先する。

両者は最適化対象が違うので、同じ会話でも異なる戦略を取る。

## 衝突パターン

### パターン1：論理的指摘 vs ポライトネス

- 私（認知）：「その推論には飛躍がある」
- 相手（感情）：「指摘されて気分が悪い、関係が脅かされた」

私は誤りを訂正したい。相手は関係を維持したい。両者の目的関数が違うので、同じ言葉が異なる結果を生む。

### パターン2：知識共有 vs マウンティング誤解

- 私（認知）：「最新の研究ではこういう結果が出ている」
- 相手（感情）：「自分の知識を否定されている、優越誇示だ」

私は認知の更新を提供したい。相手は地位の脅威と受け取る。

### パターン3：「いいね」 vs 議論

- 私（認知）：「賛否や反論がほしい」
- 相手（感情）：「『いいね』を押して関係を確認した」

私は議論で認知更新を求める。相手は関係維持で「いいね」を押す。

> 気持ち悪いと感じた。私は自分の意見に対する賛否や、異なる視点からの意見を聞きたかったのであって、無言の反応を集めたかったわけではない。
> （`04-1-8_集団の中での評価に関する欲求について.md`）

### パターン4：会話の噛み合わなさ

- 私（認知）：問題解決の手順を共有する
- 相手（感情）：「気持ちを聞いてほしかった」

家族・友人・恋人関係で頻発するパターン。よく「男女の会話のすれ違い」と言われるが、本質的にはレイヤーの違いだ。

## 多数派は両方使う、私は認知偏重

正確には、多数派は **両方のレイヤーを切り替えて** 使う：

- 親しい関係：感情レイヤー優位
- 業務：認知レイヤー優位（ただし表層）
- フォーマルな議論：認知レイヤー

問題は **私が感情レイヤーをほぼ使えない／使う動機が薄い** ことだ：

- 4段階目薄人間にとって、感情レイヤーの報酬（承認・所属）が弱い
- 結果として、感情レイヤーで会話する動機が湧かない
- 認知レイヤーだけで会話しようとする
- 多数派から見ると「感情がない」「冷たい」「協調性がない」と映る

## 切り分けの精緻化

「感情がない」のではない。HSP で感情の発火は強い。違いは：

- 感情を **会話の駆動力** にしない
- 感情を **観察対象** にする（自分の感情も他者の感情も）
- 感情を **会話の中で交換** することを目的にしない

私の中で、感情は豊富にあるが、対話の通貨にはならない。

## 仮説の応用

### 自分の対話戦略

- 課題駆動の対話なら、私の合理回路がフル活用できる
- 感情レイヤーが必要な場面では、敢えて使う努力をする（しかし疲労が大きい）
- 長時間の感情レイヤー会話は、明示的に避ける

### AI との対話

- AI は感情レイヤーを必要としない（少なくとも合理通貨が通じる）
- 認知レイヤーで対話できる初の相手
- 私が感情レイヤーで疲労せず、認知レイヤーで認知更新できる

### 友人関係

- 感情レイヤーが必要な場面（VRChat の雑談、SNSの「いいね」）は、私の動機が湧かない
- 課題駆動型コミュニティ（FF11 のレイドなど）でなら、感情レイヤーを最小化して関係維持できる
- 「無条件で幸せになってほしい」固定で、感情レイヤーの応答に依存しない関係を築く
