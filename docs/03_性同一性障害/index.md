# 3. 性同一性障害

## この章について

3つの生得的特性のうち、**性同一性障害（メイレズビアン）** について、関連するエピソードから分析する。

用語そのものの説明は [性同一性障害とは](../性同一性障害とは.md) を参照。

47歳のときに、自分が性同一性障害（生物学的男性・性自認女性・性的指向女性 = メイレズビアン）であると気づいた。それまで47年間、自分が他人と何かが違うと感じていた違和感が、この一つの認識で氷解した、というのが本人の感覚（[EP-040](../エピソード/EP-040_47歳でメイレズビアンと気づいたとき「自分の中に魂が入った」感覚.md)）。

## どう現れているか

性同一性障害は、私の人生のさまざまな場面に現れている。エピソードを系統別に並べる。

### 身体への現れ

身体に関わる過敏さ・違和感として現れたもの。

- [EP-001: 海水パンツでハイレグでへそを隠す](../エピソード/EP-001_海水パンツを引き上げてハイレグでへそを隠していた.md)
- [EP-002: 走り方が「女の子みたい」と何度も言われた](../エピソード/EP-002_走り方が「女の子みたい」と何度も言われた.md)
- [EP-004: 身体測定で他人に素肌を触られて本気で怒った](../エピソード/EP-004_身体測定で他人に素肌を触られて本気で怒った.md)
- [EP-005: 銭湯・温泉に強い違和感がある](../エピソード/EP-005_銭湯・温泉に強い違和感がある.md)

これらに共通するのは、自分の身体を「見られる対象」「触れられる対象」として強く意識し、過敏に反応していたことだ。脳の性と身体の性が一致しないため、本能的に女性的な身体感覚に近い形で、自分の体を守ろうとしていたのかもしれない（EP-001、EP-005の本人の考察より）。

### 性嗜好への現れ

- [EP-006: エロ本でレズビアンものに最も強く反応した](../エピソード/EP-006_エロ本でレズビアンものに最も強く反応した.md)

媒体が変わっても性的趣向は一貫していた。色々と試した結果、おそらく自分は受け側の女性的な感覚が好きなんだという結論に至った、というのが本人の整理。

### 物語選好への現れ

物語の登場人物への感情移入の仕方にも、性自認の影響が出ている。

- [EP-017: 中学時代に女性主人公のラノベを無意識に手に取った](../エピソード/EP-017_中学生の頃、女性主人公のラノベを無意識に手に取った.md)
- [EP-025: 格闘ゲームで一貫して女性キャラを選んだ](../エピソード/EP-025_格闘ゲームで一貫して女性キャラを選んだ.md)

「これは私が性自認を女性として持ち、恋愛的な指向も女性に向いていることと深く関係していたのだと気づいた」（EP-025原本）。

### 自己像・アイデンティティへの現れ

社会の中で自分が何者かが定まらなかったことに由来する、自己評価のねじれ。

- [EP-011: 「うまい？」と聞かれて「おいしいと思います」と答えた](../エピソード/EP-011_「うまい？」と聞かれて「おいしいと思います」と答えた.md)
- [EP-039: 20代、自分を完全にダメ人間だと思っていた](../エピソード/EP-039_20代、自分を完全にダメ人間だと思っていた.md)
- [EP-040: 47歳でメイレズビアンと気づいたとき「自分の中に魂が入った」感覚](../エピソード/EP-040_47歳でメイレズビアンと気づいたとき「自分の中に魂が入った」感覚.md)
- [EP-042: 子供の頃から社会には「適応していた」のであって「参加していなかった」](../エピソード/EP-042_子供の頃から社会には「適応していた」のであって「参加していなかった」.md)

「アイデンティティを守ろうとする承認欲求が弱かったのではなく、守るべきアイデンティティそのものが長い間形成されていなかった」（EP-040原本）。47歳でメイレズビアンと自覚したことで「自分の中に魂が入った」と表現できるほどの軸が生まれた。

### 女性を見る視線への現れ

性的指向が女性であり、無意識に女性を「自分と同じ側」と捉えてしまっていたことから派生した嫌悪感のパターン。

- [EP-029: 嫁姑関係のドラマがものすごく嫌いだった](../エピソード/EP-029_嫁姑関係のドラマがものすごく嫌いだった.md)
- [EP-030: 給食を残せない男子に女子が無理に食べさせる場面が嫌だった](../エピソード/EP-030_給食を残せない男子に女子が無理に食べさせる場面が嫌だった.md)
- [EP-031: 嫉妬の感情をほとんど持ったことがない](../エピソード/EP-031_嫉妬の感情をほとんど持ったことがない.md)
- [EP-032: 特定のフェミニスト女性タレントが嫌いでチャンネルを変えた](../エピソード/EP-032_特定のフェミニスト女性タレントが嫌いでチャンネルを変えた.md)

これらは「女性そのものが嫌い」というより、性的指向が女性であることから女性の在り方に対する見方が無意識に厳しくなっていた、という構造。

## 47歳の気づきの意味

47歳でメイレズビアンであると自覚するまでの間、これらのエピソードはバラバラの違和感として残っていた。自覚以後、すべてが一つの軸でつながった。

その瞬間の感覚を、本人は「自分の中に魂が入った」と表現している（EP-040）。
