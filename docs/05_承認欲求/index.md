# 5. 承認欲求

## この章について

3つの生得的特性のうち、**承認欲求の不在** について、関連するエピソードから分析する。

用語そのものの説明は [承認欲求とは](../承認欲求とは.md) を参照。

私はマズローの欲求段階説でいう4段階目の承認欲求（主に他者承認＝社会的立場獲得本能の発露）が、極めて薄い、あるいはほとんど存在しない。代わりに動いているのは、合理性と整合性で世界を判定するモード、いわゆる「合理性駆動モード」。

## 承認欲求の代わりに何で動いているか

合理性と整合性で世界を判定するモード、いわゆる **合理性駆動モード**。

- 内から出てきた欲求に対して「これは合理的か」と判断する
- 合理的でなければ時間をかけてでも手放す（煙草・パチンコをやめた）
- 合理的であれば自然にそれに従う（全体を良くしたい志向）

「自分は欲求を合理性で処理する」「多くの人は欲求を承認という回路で処理する」「立場を利用するという行動は、承認欲求の層の上で起きる。その層がない自分には、立場を道具として使う動機がそもそも存在しない」（EP-041、EP-042原本のメモより）。

## 補足：性欲・ドーパミン系は別系統

承認欲求がないからといって、すべての欲求がないわけではない。

- 三大欲求（生理的欲求）はある
- 性欲はある
- ドーパミン系の刺激欲求（煙草、パチンコ、ギャンブル系の引き）はある
- 物語への没入欲もある（HSP由来）

これらと承認欲求は別の層なので、「承認欲求がない」は「無欲」を意味しない。
