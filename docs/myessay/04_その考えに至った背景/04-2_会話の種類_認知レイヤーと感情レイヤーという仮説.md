# 4-2　会話の種類 ― 認知レイヤーと感情レイヤーという仮説

本稿では、日常の会話を理解するための枠組みとして、「認知レイヤー」と「感情レイヤー」という二層モデルを仮説的に導入する。この用語自体は特定の学問分野で公式に定義されたものではないが、以下に挙げる複数の学術的知見を統合的に整理した概念である。

第一に、認知心理学のデュアルプロセス理論がある。カーネマンらが示したように、人の思考は大きく二つの様式に分けられる。

**System 1** は直感的・感情的で高速かつ自動的に働く処理であり、日常的な判断や対人反応の多くを担う。

**System 2** は論理的・熟考的で低速な意識的処理であり、因果関係の検討や整合性の確認を行う。

本稿でいう感情レイヤーはおおむね System 1 に、認知レイヤーは System 2 に対応する。

第二に、社会的コミュニケーション研究では、会話には少なくとも二つの機能があるとされる。

一つは**関係機能（relational function）**で、安心・共感・役割調整など関係の維持を目的とする。もう一つは**課題機能（task / informational function）**で、事実伝達や問題解決など内容の処理を目的とする。この区分は、本稿の二層モデルの直接の理論的土台となっている。

第三に、言語学のポライトネス理論では、人は会話の中で自他の「フェイス（面子）」を守ろうとする傾向が指摘されている。ブラウンとレヴィンソンは、情報の正確さよりも関係維持が優先される場面が多いことを示し、論理的指摘がしばしば回避される理由を説明した。

第四に、社会神経科学の研究からは、**感情共感ネットワーク**と**認知的視点取得ネットワーク**という別系統の脳回路が存在することが明らかになっている。これは、人間のコミュニケーションが神経基盤のレベルでも二系統に分かれている可能性を示唆する。

以上の知見を踏まえ、本稿では「会話には感情レイヤーと認知レイヤーが存在する」という仮説を出発点とする。前者は関係維持と安心の交換を主目的とし、後者は事実と論理の共有を主目的とする。両者は本来補完的であるが、目的関数の違いゆえに衝突が生じやすい。本稿では、この枠組みを用いて、私が経験してきた会話の違和感とその構造を論理的に整理していく。
