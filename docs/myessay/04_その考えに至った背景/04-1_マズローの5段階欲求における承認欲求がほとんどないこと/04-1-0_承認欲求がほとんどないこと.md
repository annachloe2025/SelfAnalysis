# 4-1　マズローの5段階欲求における「承認欲求」がほとんどないこと

自分がメイレズビアンであると気づいて以降、その特性について、いくつか調べたり考えたりするようになった。その過程で、自分には「承認欲求」がほとんど存在しないのではないか、という点に思い至った。

この「承認欲求」という言葉を使うにあたって、まず前提として説明しておく必要があるのが、**マズローの5段階欲求説**である。これは、人間の欲求を段階的に整理した考え方であり、「承認欲求」もその中の一つとして位置づけられている。

以下では、私自身の話に入る前に、**Abraham Maslowが提唱した5段階欲求がどのようなものかを、簡単に整理**しておきたい。そのうえで、なぜ私は自分に承認欲求がほとんどないと感じるようになったのか、順を追って説明していく。
