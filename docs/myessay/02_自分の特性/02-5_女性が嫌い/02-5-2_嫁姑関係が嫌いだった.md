# 2-5-2 嫁姑関係が嫌いだった

子供の頃、ドラマなどでよく目にした嫁姑関係が、私はものすごく嫌いだった。そこに映るやり取りは、ひどく醜いものに見えた。

のちに自分がメイレズビアンで、承認欲求がほとんどないことに気づいてから、この嫌悪感の理由がよく分かるようになった。

私は、嫁姑関係の対立はまさに承認欲求の発露だと思う。そこで起こる衝突の多くは、「どちらがこの家庭や息子にとって重要な存在か」という承認ポジションの取り合いに近い。

しかし私は承認欲求がほとんどないため、その感覚がまったく理解できなかった。だから当時は、単に性格の醜い人間同士の争いにしか見えなかったのである。

さらに、私の性自認が女性で、性的指向も女性に向いていることから、女性のあり方に対する見方が無意識に厳しくなっていたのだと思う。

その結果、嫁姑争いをする女性は醜悪な存在だという極端な結論に至ってしまっていた。

現在では、多くの人が承認欲求を持っているからこそ嫁姑問題が起こりやすいのだと理解できるようになった。そのため、以前ほどこの問題に強い嫌悪を抱くことはなくなっている。
