# 2-5-4 嫉妬の感情がない

私は小学生の頃から、嫉妬という感情をほとんど持ったことがなかった。二十歳くらいの時には、そのことを自覚していたと思う。しかし当時は、その原因までは分からなかった。今振り返ると、承認欲求が希薄であったこと、そしてその背景には、性同一性障害の一種であるメイレズビアンという自己認識があり、体と心の不一致を受け入れる過程で、私の中では承認欲求がそもそも発露しなかったのだと感じている。

嫉妬の感情がないと、嫉妬している相手の姿がひどく醜悪に見えてしまう。まるで単なる悪人のように映るのだ。

なぜなら、私の目には、相手が理由もなく他人に嫌がらせをしているようにしか見えなかったからである。

多くの人は嫉妬心を持っているため、嫉妬から何か悪い行動を取る人を見ても、「嫉妬心の強い人だ」くらいにしか思わないのだろう。しかし、そもそも嫉妬心がない側からすると、それは感情の問題ではなく、ただ悪いことをしている人間にしか見えない。

女性は比較的嫉妬を抱きやすいと言われるが、当時の私には、そうした姿が単に「女性は悪人だ」という極端な印象に結びついてしまっていた。また職場などで、嫉妬のあまり優秀な人の足を引っ張る人を見ると、それもまた純粋な悪意の行為にしか思えなかった。

こうした状態が二十代くらいまで続いたため、私の考え方はかなり歪んでいたと思う。

多くの女性を嫌いになり、真面目に働かない人間にも強い嫌悪感を抱いていた。本来なら、「人には誰にでも嫉妬心があり、それが抑えられているかどうかの違いなのだ」と理解できたはずだ。しかし、私自身に嫉妬心がなかったため、その視点を持つことができなかった。

この歪みは、女性と付き合いたいと思えなくなったり、仕事を長く続けられなくなったりしたことの大きな要因の一つだったのではないかと思う。

今になって振り返ると、承認欲求がないということが人生に与える影響は、想像以上に大きかったのだと感じている。
