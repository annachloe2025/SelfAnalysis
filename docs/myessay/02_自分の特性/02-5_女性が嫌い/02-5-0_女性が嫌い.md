# 2-5 女性が嫌い

私は小学生の頃から、なぜか「女性」という存在に対して強い嫌悪感を抱くことがあった。もちろん、全ての女性が嫌いだったわけではない。優しい人もいれば、尊敬できる人もいた。それでも当時の私は、「女性に多いと感じた特定の振る舞い」や「女性同士の関係性」に触れるたびに、強い不快感が立ち上がってしまった。

直接的なきっかけとして思い出せるのは、いくつかある。

小学生の頃、年上の女性から「一人っ子だから我儘ね」といったことを何度も言われたことがある。私はその言葉がとても嫌だった。私の内面を見ずに、属性だけで決めつけられ、軽く裁かれたように感じたからだ。

また、テレビなどで見た嫁姑の関係——意地悪、支配、見下し、陰湿さのようなもの——が、強い不快感として残っている。今思えば、当時の私が「人を痛めつける関係性」そのものに敏感だったのだと思う。

学校では、いじめられている男子に対して、周囲の女子が「給食を残すな」と無理に食べさせようとする場面があった。本人が困っているのに、助けるのではなく"正しさ"で押し切る。私はそこに、支配のようなものを感じてしまい、とても不快だった。

そして何より、女性の嫉妬が怖く、苦手だった。嫉妬が表に出るときの空気、遠回しな攻撃、仲間内の圧力——そうしたものが私には重く、身構えさせるものだった。

大人になってからも、その感覚は完全には消えなかった。特定の女性タレントがどうしても受け付けず、テレビに映るだけでチャンネルを変えるほど嫌悪することもあった。

不思議だったのは、私は本来、あまり人の好き嫌いをしない性格だったことだ。しかも、理由もなく他人を嫌うという行為そのものを、どこかで嫌っていた。そんな自分が、なぜこれほどまでに「女性」に反応してしまうのか——当時の私はその原因が分からなかった。

その頃の私は、短絡的に「女性は性格が悪い人が多いのだ」と結論づけていた。しかし今なら、それは事実というより、私の側の問題——私の感じ方の癖や、私の内側の事情——が大きく関わっていたのだと分かる。

そして、自分がメイレズビアンであると自覚してから、この嫌悪感の正体が少しずつ見えるようになった。

私は「女性そのもの」を嫌っていたのではなく、女性をめぐるさまざまな刺激が、私の中の何かを強く揺さぶり、苦痛や恐怖や反発を引き起こしていたのだと思う。

ここから先は、当時の具体的な出来事を手がかりにしながら、「なぜ私は女性に強く反応してしまったのか」を一つずつ整理していきたい。
