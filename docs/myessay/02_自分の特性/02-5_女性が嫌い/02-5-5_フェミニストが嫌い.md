# 2-5-5 フェミニストが嫌い

私が二十代の頃、テレビには女性のフェミニストが頻繁に出演していた記憶がある。当時の私は、彼女たちがとても苦手だった。

その理由について、あらためて考えてみたい。

当時の私の目には、彼女たちはあらゆる話題を女性問題に結びつけ、最終的には「男性が女性を差別しているから悪い」という結論へと導いているように映っていた。しかもそれを、あたかも疑いようのない正論であるかのように、強い怒りを伴って語る。その話し方自体が、一つのテンプレートのように感じられた。

今でも、こうした話し方には違和感を覚えるし、おかしいと感じている。しかし、なぜ当時のテレビでは、このタイプの人々が多く取り上げられていたのだろうか。おそらく、そこには世の中の需要があったのだと思う。そして多くの人は、この話し方を特別おかしいとは感じなかったのだろう。

多くの人が違和感を覚えない理由として、私は「承認欲求」の存在が大きいのではないかと考えている。多くの女性が承認欲求を持っている場合、あらゆる話題を女性に関連づけ、「男性差別が原因だ」という論調は、感情的に有利に働きやすい。共感を得やすければ視聴率も上がるし、当時フェミニズムが一種のブームであったことも重なって、テレビで扱われる機会が増えたのだろう。

一方で私は、承認欲求がほとんどなかったため、この話し方に強い違和感を覚えた。私にはそれが、論点を意図的にずらした詭弁のように見え、事実よりも感情を優先して世の中を扇動しようとしているように感じられた。そのため、その女性タレントがテレビに映るだけで、チャンネルを変えてしまうほど嫌悪感を抱いていたのだと思う。

さらに、私自身がメイレズビアンであり、女性に対して厳しい視線を向けやすい性質を持っていたことも、こうした感情を加速させていたのだろう。

周囲の人に「ああいうタレントが嫌いなんだよね」と話しても、あまり共感は得られなかった。そのことからも、自分の感じ方が少数派であったことは薄々分かっていた。

現在では、当時の自分がなぜそう感じていたのかを、ある程度言語化できるようになった。今でも、あのような語り口のフェミニストは好きではない。しかし同時に、本来のフェミニズムとは、ああした感情的で一面的な語り方をするものではない、ということも学んだ。

その意味では、もはや「フェミニストが嫌い」というよりも、「特定の語り方や姿勢が嫌いだったのだ」と理解するようになっている。
