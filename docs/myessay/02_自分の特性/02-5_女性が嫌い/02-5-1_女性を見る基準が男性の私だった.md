# 2-5-1 女性を見る基準が男性の私だった

ここから女性を嫌いになってしまった理由を説明する前に、私の性のあり方と、そこで起きていた心理的な仕組みを整理しておきたい。

私は生物学的には男性である。しかし性自認は女性で、性的指向も女性に向いている。身体・心・恋愛の方向が互いに食い違った状態にあった。

人はふつう、男性同士なら「同じ側の人間」と感じ、女性に対しては「自分とは違う性」として距離を取りながら理解しようとするのではないだろうか。異性はどこか感覚の違う他者として認識されやすい。

しかし私の場合、心の中では自分を女性だと捉えていたため、女性を「自分と同じ側の存在」と無意識に感じていた。身体は男性でありながら、その自覚がほとんどないまま、脳は自分を女性として処理していたのである。

ここに第一の混線があった。
本来なら「異なる他者」として向き合うはずの女性を、私は「自分と同じ価値観を持つはずの人間」として見てしまっていた。

心理学では、人は恋愛対象に対して自分の感覚や価値観を重ねやすいとされ、これを「投影」や「理想化バイアス」と呼ぶ。さらに、自分に似ているはずだと思い込む「類似性の思い込み」も働きやすいという。私の中では、これらがかなり強く起きていたのだと思う。

私にとって好意の対象は女性であり、しかも自分も女性側の人間だと感じていた。ところが実際の身体や経験は男性である。そのため、自分の内側にある男性的な感覚や価値観を、好意の対象である女性全体に無意識に投影してしまっていた。

そこにもう一つの要因が重なる。私は承認欲求がほとんどなく、嫉妬や妬み、野心といった感情が弱い。だからこそ、そうした感情を持つ人の姿が理解しにくく、それもまた女性全体への評価に重ねられていった。

この二つが合わさるとどうなるか。
多くの女性は、私の中で作られた理想像から大きく外れた存在に見えてしまう。承認欲求やネガティブな感情を持つことは人として自然なのに、私はそれを受け止めきれず、醜いもののように感じ取っていたのである。

こうした認知のねじれ——同一視、投影、理想化——が無意識のうちに積み重なった結果、私はほとんどの女性に対して強い拒否感を抱くようになった。ただ、もともと他人を理由なく嫌うことに抵抗があったため、表立って敵意を向けることはなかった。それでも関係が近づくほど、恋愛対象としては受け入れられなくなっていった。

しかし、自分がメイレズビアンであると理解してから、この状態は大きく緩和された。
私の中で何が起きていたのかを言葉にできたことで、長年の誤りにようやく気づくことができたのだと思う。
