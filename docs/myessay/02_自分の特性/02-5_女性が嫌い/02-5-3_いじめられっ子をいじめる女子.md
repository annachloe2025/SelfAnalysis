# 2-5-3 いじめられっ子をいじめる女子

学校では、いじめられている男子に対して、周囲の女子が「給食を残してはいけない」と言って無理に食べさせようとする場面があった。本人が明らかに困っているのに、助けるのではなく"正しさ"で押し切る。その光景に、私は支配のようなものを感じ、とても不快だった。

その男子は吐きそうになりながら嫌いな給食を口に運んでいた。それを女子が「残しちゃだめ」と言って食べさせ、先生もそれを容認している――そんな構図だった。私はこの状況がものすごく気持ち悪く、強い嫌悪感を覚えた。

この件については少し複雑で、自分の解釈に完全な自信があるわけではないが、なぜ私がそこまで嫌悪したのかを整理してみたい。

まず、私はHSPであるため、いじめられている側に強く共感しやすい。えづきながら食べている男子の姿を見て、ほとんど自分のことのように苦しさを感じ、この出来事を「いじめ」に近いものとして受け取ったのだと思う。

次に、この場面は承認欲求が生み出す社会の縮図のようにも見えた。女子は同調圧力の実行者であり、先生は権威による承認者、男子は逃げ場のない個人――まるで社会で起こる上下関係や支配構造が、そのまま教室の中に再現されているようだった。

そして私は承認欲求がほとんどないため、この構図そのものが直感的に理解できない。多くの人にとっては「よくある指導」「普通のこと」に映るのかもしれないが、私には単なるいじめにしか見えなかった。

さらに、周囲のクラスメートがこの状況を特に問題視していないことが、私の違和感をいっそう強めた。おそらく多くの人は承認欲求のフィルターを通してこの場面を受け止めるため、あるいはHSPではないために男子の苦痛にそこまで共感せず、意識に上らないのだろう。結果として、私だけがこの出来事を深刻に受け取っているように感じられ、それが孤立感につながった。

加えて、前章で述べたように、私はもともと女性に対して嫌悪感を抱きやすい体質であり、そのためこの女子の行動が実際以上に冷酷でひどいものとして映ったのだと思う。

これらの要素が複雑に絡み合い、結果としてこの出来事は私の中に強く残る嫌な記憶となった。この問題に限らず、同じような構造で私だけが強い不快感を抱く場面は、これまでにも何度もあった。

以前は、世の中の八割くらいの人はどこかおかしいのではないか、と本気で考えたこともある。しかし今は、私がHSPで承認欲求が弱く、さらにトランスジェンダーという少数側の立場にあり、その中でもメイレズビアンというさらに少数の存在であることが、この感じ方の大きな要因だったのだと理解している。
