# 2-3 女性キャラに感情移入する

私は高校生の頃、格闘ゲームをよく遊んでいた。友人とゲームセンターに行くことも多く、そのとき私はプレイキャラとして女性キャラクターを選ぶことが多かった。女性キャラばかり使っていると周囲の目が少し気になり、意識して男性キャラも使うようにしていたが、本当は女性キャラを操作するほうがずっと楽しかった。

小説やアニメについても、女性キャラクターが主人公の作品に強く惹かれていた。中学生の頃に無意識で選んだライトノベルには女性主人公のものが多く、好きなアニメや漫画も同じ傾向があった。

後になってから、これは私が性自認を女性として持ち、恋愛的な指向も女性に向いていることと深く関係していたのだと気づいた。自分の内面の在り方を理解するにつれ、なぜ女性キャラクターにこれほど自然に感情移入していたのかが腑に落ちていった。

その結果として、物語の中でも女性キャラには強い親近感を抱きやすく、男性キャラにはあまり関心を向けないという傾向が生まれていたのだと思う。以前はそれがほとんど無意識の反応だったが、自分の在り方を自覚してからは、よりはっきりと表れるようになった。同時に、自分が心から楽しめる作品も以前より選びやすくなった。
