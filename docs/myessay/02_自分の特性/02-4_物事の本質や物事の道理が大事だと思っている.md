# 2-4 物事の本質や物事の道理が大事だと思っている

私は小学生の頃から、物事の本質や道理をとらえることが大事だと考えてきた。

単なる丸暗記のような学び方にはあまり意味がなく、その物事がなぜそうなっているのか、どのような理屈で成り立っているのかを理解することのほうが重要だと思っていた。

もしかすると単純に暗記が苦手だっただけかもしれないが、暗記に頼るくらいなら、道理や仕組みを理解しようと努力していた記憶がある。

なぜこうした考え方をするようになったのかを振り返ると、いくつか理由が思い当たる。

まず一つは、DNA検査で分かった「開放性が高い」という気質である。新しい考えや抽象的な概念に関心を持ちやすい性質が、生得的に備わっていたのだと思う。

また、私はHSPの傾向があり、物事を深く考えやすい体質だと言われる。詳しいメカニズムは分からないが、感受性が敏感なため、普通の人より多くのことが気になり、それが思考を促していたのではないかと思う。

さらに、承認欲求がほとんどなかったことも大きい。子どもの頃から「生きている意味とは何だろう」と考えることが多く、生きるための動機が周囲の人より少なかったのだと思う。そのため、自分なりに生きる理由や価値を作り出そうとしていた。当時は自覚していなかったが、今振り返れば、明らかに「生きるための動機づくり」をしていたのだと分かる。

こうした背景から、物事の道理や本質に強い関心を持つようになり、思考も自然とそちらに偏っていったのだと思う。

二十歳を過ぎた頃からは、起きている間はずっと何かを考えている感覚になった。無意識に思考が続くこの状態はDMN、デフォルトモードネットワークと呼ばれるらしい。歩いている時も、車を運転している時も、風呂に入っている時も、常に頭の中で何かを考えていた。物事の道理や本質は何なのかを、ああでもないこうでもないと考え続けていたし、悩みがある時は同じ考えが延々とループすることもあった。

こうした思考の癖が、三十代でうつ病になった一因であることは間違いないだろう。また、便秘症になってしまったのも、この過剰な思考と無関係ではない気がしている。

この思考傾向は、私の人生において非常に大きな位置を占める特性だと思う。今こうして文章を書いているのも、結局はこの性質があったからなのだろう。
