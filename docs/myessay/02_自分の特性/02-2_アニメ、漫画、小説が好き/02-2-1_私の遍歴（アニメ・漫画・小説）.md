# 2-2-1 私の遍歴（アニメ・漫画・小説）

私は、アニメ、漫画、小説といった物語が好きだ。子供の頃から関心はあったものの、習い事のため夜のアニメの時間に家を空けていたことや、シングルマザーの一人っ子で周囲から漫画やアニメの情報が入りにくかったこともあり、今ほど多くの作品に触れてはいなかったと思う。

中学生の頃、なぜか国語だけが苦手だった。これを改善したいと思い、自分から小説を読んでみようと考えたのがきっかけでライトノベルを読み始めた。特に知識もないまま書店で自分の感覚だけを頼りに選んでいたが、『スレイヤーズ』や『燃える瞳のメル』など、女性主人公の作品を無意識に手に取ることが多かった。性自認が女性であることを考えると、その選択にも自然な理由があったのだと今では理解できる。

高校に入ると、クラスでジャンプやマガジンなどの少年誌を回し読みする風習があり、それをきっかけに漫画にも触れるようになった。その頃、自分は周囲より読むのが少し遅いのではないかと感じていた。おそらく文字を読むこと自体がやや苦手だったのだと思う。後にDNA検査で情報処理がやや遅い傾向が示されたことからも、ある程度納得がいった。

専門学校に進学してからは、友人の勧めで推理小説を読むようになった。森博嗣や京極夏彦の作品が好きで、よく手に取っていた。そこで初めて「頭のいい人はこういう思考をするのか」と感じ、自分でもいろいろなことを学びたいと思うようになった。

高校の終わり頃には『エヴァンゲリオン』が流行し、それを機にアニメをかなり見るようになった。物語のテーマ性や世界観についても考えるようになり、最初は理解できなかった部分も、何年かすると自然と腑に落ちるようになっていった。

30代に入ると、漫画やアニメにあまり関心のない人が意外に多いことに気づいた。多くの人が物語のテーマや世界観を深く理解しようとしていないことにも驚いた。もっとも、私自身も20代前半までは十分に理解できていなかったのだと思う。この頃、自分の認知のレベルが少しずつ上がっていることを実感した。

20代後半から30代後半にかけては強いストレスに悩まされ、うつ病にもなった。その時期、アニメを見ていると気持ちが楽になった。物語を追っている間は余計なことを考えずに済み、結果としてストレスが軽減されたからだと思う。その影響もあり、この頃から非常に多くのアニメを見るようになった。

最近になって、46歳頃からオーディブルの存在を知り、聴きながら仕事や生活をするようになった。これは驚くほど効果があり、一日中楽しい気分で過ごせるようになった。数か月間、ほとんど途切れることなく心が満たされた状態が続いた。Kindleの読み上げ機能でも同じような感覚を得られた。しかし、問題もある。心から面白いと感じる作品でオーディブル化されているものはそれほど多くない。また、常に「楽しい状態」でいると、新しいことを学んだり他の課題に取り組んだりする意欲が薄れてしまう。適度に利用しなければ、生活習慣そのものが崩れてしまう危険もあると感じている。
