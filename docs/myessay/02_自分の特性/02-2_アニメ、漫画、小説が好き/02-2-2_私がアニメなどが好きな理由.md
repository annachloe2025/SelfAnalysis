# 2-2-2 私がアニメなどが好きな理由

世の中の多くの人が、私と同じようにアニメなどが好きであると思っていた。

しかし、現実は少し違っていて、多くの人は私のようにアニメを楽しんではいなかった。

私の場合、ややアニメ制作者のようにアニメを楽しむ傾向があると思う。例えば、このアニメで伝えたいテーマを何かを考えたり、このシーンはどういう印象を受けるかを感じたり、その表現方法がどういう方法だからそういう印象を受けるのか考えたり、世界観の構築の理由や世界観から感じるイメージを感じたり考えたり、キャラのセリフから感じる情動とそれはどうやって表現されているのかなどを感じたり考えたりするのがすきだ。

一方で、多くの人は作品の一部分を中心に楽しんでいるように感じる。物語そのものに深く入り込むというより、印象的な場面や分かりやすい魅力だけを拾い上げているように見えることがある。

また、アニメを話題づくりの手段として利用している人も少なくないように思う。誰かと会話をするための共通の材料として作品を見て、その感想をやり取りする——そうした関わり方だ。

それはおそらく、作品を味わうこと以上に、他者とのつながりや評価を得ることに比重が置かれているからではないかと感じる。私には、本質としてのアニメというより、承認を得るためのきっかけとしてのアニメに近いように映ることがある。

最初は、みんなアニメに対する理解が浅いとかちゃんと考えていないとか思っていた。しかし、自身に承認欲求がないと知ってからは、承認欲求を満たすためのアニメの利用法とかもあるかもしれないと思い始めた。また、自身がHSPと気が付いてからは、多くの人はアニメで視聴者に伝えようとしている演出はHSPである自分ほど、多くの人は感じ取れていないのではないかと思うようになった。

つまり、承認欲求がないので、承認欲求目的でのアニメの活用がない、HS'Pの影響でより多く影響をアニメから受ける、の理由で普通の人より私は、アニメが好きになりやすい体質なのである。
