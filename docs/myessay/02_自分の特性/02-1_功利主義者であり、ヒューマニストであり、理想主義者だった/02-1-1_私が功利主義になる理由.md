# 2-1-1 私が功利主義になる理由

高校を卒業したあたりから、私は「会話は有益であるべきだ」という意識をはっきり持つようになった。私と話すことで、相手が少しでも成長したり、何かを得たりできたらいい——そんな思いで人と接していた。

同じ頃から、「できるだけ多くの人の幸福を、できるだけ大きくしたい」という考えを持つようになった。後になって知ったが、これは功利主義と呼ばれる考え方に近いらしい。

私がこうした発想を持つようになった背景には、自分の性のあり方と、承認欲求がほとんどないという性質が大きく関係していると思う。私は生物学的には男性だが、性自認は女性で、性的指向も女性に向いている。しかも、多くの人が自然に持つ「認められたい」「注目されたい」「他人より上に立ちたい」といった承認欲求が、私にはほとんどなかった。

多くの人は、ヘテロセクシャルで承認欲求を土台にしながら、あまり迷わず本能に従って成長していくのだと思う。しかし私の場合、小学生の頃から「自分は何のために生きているのか」をよく考えていた。おそらく、普通なら自然に生まれるはずの欲求が、私の中ではうまく機能していなかったのだと思う。

性欲そのものは発達していたが、それが男性として女性を求める形ではなく、レズビアン的な方向でしか表れなかった。その時点で現実の生活と結びつきにくく、思春期の成長過程で大きな混乱が起きても不思議ではなかった。

つまり、普通の人が持つような「自分の内側から湧き出る人生の目的」が、私にはなかった。だからこそ、生きる意味を必死で考え、頭の中で一から組み立てていくしかなかったのだと思う。

その過程で、「人はこう生きるべきだ」という規範のような考え方が強くなっていった。戦争はすべきではない、だから平和につながる行動を取るべきだという発想や、マザー・テレサのような生き方こそ正しいという感覚である。

そうした流れの先に、「有益な会話をして友人たちのレベルアップに役立ちたい」「世界全体の幸福度を高め、痛みをできるだけ減らすべきだ」という考え方が、私の中で自然に"正解"のようになっていった。

多くの人は、おそらくそこまで考える必要がない。自分の内側から出てくる承認欲求に従って行動すれば、生き方は自然に決まっていくからだ。たとえば、モテたい、認められたいという動機から、お金を稼ごうとしたり、出世を目指したりする——それで人生は十分に回っていく。

頭のいい人の中には、思考の結果として功利主義にたどり着く人もいるだろう。しかし私の場合は、承認欲求の乏しさと性のあり方の問題によって、本能的な指針を持てず、理屈を拠り所にするしかなかったというのが実感に近い。

こうした事情が重なった結果、私は功利主義に近い考え方を持つようになったのだと思う。
