# 2-1 功利主義者であり、ヒューマニストであり、理想主義者だった

私は小学生の頃、尊敬する人はマザー・テレサで、「人はみんなマザー・テレサのように生きるべきだ」と本気で思っていました。

また、どうすれば世界が平和になるのか、いじめがなくなるのかといったことも、その頃からよく考えていました。

権力を振りかざすような人は嫌いでしたし、「偉い人＝お金持ち」という意見にも強い違和感がありました。お金や権力を持っているから偉いという考え方は、まったくの間違いだという感覚が強かったのです。

高校を卒業したあたりからは、「会話は有益であるべきだ」という意識を持つようになりました。私と話すことで、相手が少しでも成長したり、何かを得たりできたらいい、そんな思いで人と接していました。

そして私は、できるだけ多くの人の幸福を、できるだけ大きくしたいと考えていました。これは功利主義と呼ばれる考え方だそうです。

さらに、人は本来大切にされるべき存在であり、いじめや差別、抑圧はおかしい、弱い人を踏みつける社会は間違っているとも考えていました。これはヒューマニズムという立場に近いようです。

加えて、小学生の頃から世界平和を思い描き、「人は善く生きるべきだ」「会話は有益であるべきだ」と信じていた私は、理想主義者とも呼べるのだと知りました。

つまり私は、功利主義であり、ヒューマニズムであり、理想主義者だったのです。

さらに若い頃の私は、世の中の多くの人も基本的には同じような考え方をしていると思っていました。みんな心の底ではこうした価値観を持っていて、努力家や真面目な人はそれを実行し、怠惰で不真面目な人は実行していないだけだと考えていたのです。

二十代前半くらいまではそう信じていましたが、そこから次第に「何かおかしい」と感じるようになりました。そして、多くの人は必ずしもこのような考え方をしているわけではないのだと結論づけたのは、おそらく四十歳近くになってからだったと思います。

そして四十七歳のとき、自分には承認欲求がほとんどないという事実に気づき、ようやく腑に落ちました。

それらについては、これから順に説明していきます。
