# 2-1-3 私が理想主義者になってしまう理由

## 私が理想主義者になってしまう理由

私は小学生の頃から世界平和を思い描き、「人は善く生きるべきだ」「会話は有益であるべきだ」と信じていた。後になって、こうした考え方は理想主義と呼ばれるものに近いのだと知った。

私が理想主義者になりやすい理由の一つは、承認欲求がほとんどないことにあると思う。

多くの人は承認欲求を持ち、それに沿って生きている。自分の立場を良くしたい、褒められたい、注目されたい、時には他人に嫉妬する——そうした感情が行動の動機になっている。

しかし私には、その感覚があまりない。出世したいとも、褒められたいとも思わないし、目立ちたいとも感じない。他人に嫉妬することもほとんどない。

承認欲求が前提にないと、理想的な社会は比較的簡単に実現できそうに見えてしまう。もし多くの人に承認欲求がなければ、社会全体で合理的な選択をすることはそれほど難しくないのではないか、と感じるのだ。

戦争の大きな原因は食料や資源の不足にあると言われる。確かに、生存に関わる下位の欲求が満たされなければ、承認欲求がなくとも争いは起こりうる。しかし現代では、技術的には食料や資源の不足はかなり解消されており、問題の多くはそれらが不均等に分配されていることに由来しているように思う。

もちろん、承認欲求が強く働く社会では、資源を均等に分配することは簡単ではない。けれど承認欲求がなければ、自分だけを優位にしようとも、他人を妬ましく思うこともない。全体に合理的に分配され、自分の分も他人の分も確保されると分かれば、それで自然に納得できる。不平等な分配が続けば問題が起きることは明らかであり、それを避けるための対策を取るべきだと感じるだけだ。

だからといって、承認欲求が不要だと言いたいわけではない。承認欲求は大きなモチベーションであり、それがあったからこそ社会はここまで発展してきた面もあるだろう。仮に承認欲求がまったくなければ、戦争は起きにくくなるかもしれないが、同時に社会の活力も弱まっていた可能性が高い。

この話の結論は、承認欲求の是非ではない。

私自身がこうした感覚を持っているために、結果として理想主義者になりやすいということだ。そして、承認欲求が弱い人ほど、理想主義的な考え方に近づきやすいのではないか——それが私の一つの仮説である。
