# 2-1-2 私がヒューマニズム的な立場に近くなる理由

私は、人は本来大切にされるべき存在であり、いじめや差別、抑圧はおかしい、弱い人を踏みつける社会は間違っていると考えてきた。こうした感覚は、ヒューマニズムと呼ばれる立場に近いように思う。

私がこのヒューマニズム的な考え方を自然に取ってしまう理由には、HSPであることが大きく関わっている気がする。感受性が強いために、他人の痛みや不当さをより直接的に感じ取ってしまうのだと思う。

また、子供の頃に受けた日本の教育の影響もあったかもしれない。私は1977年生まれで、1980年代から90年代初めにかけての昭和の終わり頃に少年期を過ごした。その時代の日本では、反戦の価値観が社会全体に根付いており、多くの人が戦争に強い拒否感を抱いていた。毎年お盆に放送されていた『火垂るの墓』のような作品は、その象徴だったと思う。

HSPである私は、そうした教育や社会の空気から、より強い影響を受けたのかもしれない。その結果として、戦争やいじめをなくしたいという思いを自然に抱くようになり、マザー・テレサのような生き方こそ最も尊いものだという印象を持つようになった。

これらを総合すると、私の根底にはヒューマニズムに近い感覚が深く根付いているのだと思う。そして現在でも、「そうあるべきだ」という感覚が心の底から湧き上がってくる。

若い頃は、誰もが同じような価値観を持っているものだと思っていた。しかし年月を重ねるにつれ、必ずしも皆がヒューマニズム的な感覚を共有しているわけではないと感じるようになった。そして、それは社会の常識というよりも、自分自身の特性なのだと気づいた。
