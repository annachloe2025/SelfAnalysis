# 2-6 一人っ子だから我儘だと思われていた

私は子供の頃よく大人の女性から兄弟はいるのと聞かれて、一人っ子ですと答えると、では我儘ねという感じで言われることが多かった。

その為、私はこれがコンプレックスとなり、極力人に助けを求めなくなったし、色々な面で遠慮もするようになった。わがままだと思われたくなかったからだ。

しかし、そういった感じで生きているとどう考えても自分より他の人達の方が我儘だと感じる事が多かった。私が我儘だと言ってる大人の女性でさえ私より我儘なんではないだろうかと思うようになっていた。

今になって考えてみれば当然で、私は承認欲求がないので、普通の人より欲求が少ない。だから、叶えたい欲求がそもそも少ないのだ。だから我儘になりにくい体質なのである。

その状態で、自身は我儘であるから、それらを律して生きなければいけないと思って修正した結果、過度に他人に頼らなくなったり、欲しいものを口にしなくなったと思う。

30代ぐらいでうつ病になるが、その原因の一端がこの考え方な気がする。自身でもわかるぐらい自信を抑圧していたと思う。
