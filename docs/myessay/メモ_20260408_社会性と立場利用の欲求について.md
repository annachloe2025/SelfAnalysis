# 社会性と立場利用の欲求について

**日付：** 2026年4月8日
**分類：** 未整理メモ（構成3-3「承認欲求＝立場を利用する生き方」に関連）

---

## きっかけ

承認欲求という概念について、「立場を利用して何かを成そうとする」という部分に深く関わっていると感じた。

## 理解できない3つの行動

- SNSで写真をアップしたりツイートしたりすることを楽しいと思う感覚
- 会社で出世したいと思うこと
- 偉そうな態度で人と接すること

以前、マズローの承認欲求がこれに関わっていると考えた。今でもかなり関わっていると思うが、完全一致かどうかはわからない。

## 核心の気づき

**私は、社会的でありながら、自身の立場を社会的によくしようと思わない。**

- 社会性はある。みんなが幸せに生活できたらいいと思う。協力していい社会を作り、楽に安全に生きたいと思う。そういったアイデアもたくさん考えるし、実行していきたい。
- しかし、どう考えてもみんな幸せになれることを、承認欲求などの自分の立場を利用して有利に事を運ぶことを優先して壊してしまう人がかなり多い。これが全く理解できない。

## 「みんなもそう思っている」という誤解

- 以前は、全体を良くすることが合理的だから、当然みんなそう思っていると信じていた。
- ところがそうではなかった。多くの人は、合理的かどうかではなく、内側から湧いてくる欲求（承認欲求など）に従って動いていた。
- 大抵の人は「みんな幸せになればいい」とは思っていても、そうなるように考えたり行動したりはしない。考えが及ばない人もいれば、自分が幸せになることを優先して全体が見えていない人も多い。
- 地球全体として良くするべきという考えの人はあまりいないと気づいた。

## 社会との関係：参加ではなく適応だった

- 子供の頃から社会には「参加している」感覚ではなく、「社会とはそういうものだから合わせていた」という感覚だった。楽しいと思ったことはない。
- 社会は「自然にいる場所」ではなく「攻略すべき対象」だった。
- 本質を教えない先生には敵対心を持っていた。「攻略するなら仕組みを理解させてくれ」という感覚。
- メイレズビアンの自覚後、「社会に合わせなければいけない」という感覚が完全に消えた。社会から完全に抜け出した感じ。それまで自分に課していた「社会を攻略する」というミッション自体が、自分ではない誰かのために設定された目標だったと気づいた。

## 欲求の構造：承認欲求の層が欠けている

- 欲求がないわけではない。三大欲求もあるし、ドーパミン系の欲求も顕著にあった（煙草、パチンコ）。どちらも努力してやめた。
- 「全体を良くしたい」という志向も内から出てくる欲求。これはマズローでいう下層の安全欲求や三大欲求を合理的に拡張したものだと今のところ思っている。
- マズローの階層でいうと、生理的欲求があり、安全欲求もあり、自己実現的な志向もある。しかし、その間にあるはずの承認欲求の層がほとんどない。階段を一段飛ばしている構造。

## 今日の考察のまとめ

**自分と多くの人の違いは「社会性があるかないか」ではなく、「欲求と行動の間に何があるか」の違い。**

- 自分は欲求を合理性で処理する。内から出てきた欲求に対して「これは合理的か」と判断し、合理的でなければ時間をかけてでも手放す（煙草、パチンコ）。合理的であれば自然にそれに従う（全体を良くしたい）。
- 多くの人は欲求を承認という回路で処理する。「合理的かどうか」より先に「認められるかどうか」で行動が決まる。
- 立場を利用するという行動は、承認欲求の層の上で起きる。その層がない自分には、立場を道具として使う動機がそもそも存在しない。

## 整理途中の仮説

- 「立場を利用する欲求」は承認欲求そのものではなく、承認欲求を含むいくつかの欲求が混ざった現象かもしれない
- 社会性には「序列の中で自分を有利にする」型と「全体がうまくいくように協力する」型がありそう
- 自分は後者だが、なぜそうなのかはまだ完全には解明できていない

## 性同一性障害との関係（未解決）

この特性は性同一性障害と深く関わっている気がする。ただし、明確に「これが原因」とピンポイントで理解できていない。

仮説：社会の中で「立場を上げたい」と思うためには、その社会の序列に自分が自然に組み込まれている感覚が必要。性自認と身体の不一致によって、社会が用意する序列（男性としての序列）にそもそも自分を置けなかった。参加していないゲームの得点を上げたいとは思わない、という構造。実際、子供の頃から社会に「参加」していたのではなく「適応」していた。

→ この考察を続ければたどり着けそうな感覚がある。
