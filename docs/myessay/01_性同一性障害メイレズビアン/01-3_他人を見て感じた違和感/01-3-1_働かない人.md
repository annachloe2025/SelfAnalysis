# 1-3-1 働かない人

人生の戦術として、会社などの組織の中で、できるだけ仕事をせずに給料をもらおうとする人がいる。

以前の私は、なぜそういう行動を取るのかが、まったく理解できなかった。
それどころか、強い憤りを覚えていた。

仕事をしない人が増えれば、会社の業績は悪化する。
業績が悪化すれば、いずれ自分たちの給料も下がる。
そうなるのは、ほぼ確実だと思っていたからだ。

だから私は、「なぜそんなことをするのか」「自分の首を絞めているだけではないか」と考え、納得できずにいた。

後になって、自分には承認欲求がほとんどないのだと理解してから、ようやくその行動が少し見えるようになった。

多くの人は、承認欲求に従って行動している。
本人にその自覚がなくても、無意識のうちに、そう振る舞っている。

例えば、職場の人間関係の中で優位な立場を作り、仕事をあまりしなくても済む位置に収まる。
そうすることで、「得をしている」「上に立っている」という感覚を得る。
それ自体が、ある種の承認欲求を満たしているのだと思う。

その行動は、本能的には「正解」と感じられているのだろう。

ただ、会社全体で見れば、従業員同士で力関係を作り、仕事を押し付け合っても、トータルの収入が増えることはない。
会社の業績が上がらなければ、結局は意味がない。

当時の私は、このズレを理解できていなかった。
その結果、ひとりで憤り、「おかしな世の中だ」「やっていられない」と感じ、会社を辞める。
そして、同じことを繰り返していた。
