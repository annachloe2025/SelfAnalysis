# 1-3　他人を見て感じた違和感

私は、他人を見ていて、理由ははっきりしないが強い違和感を覚えることがある。

それは、特定の誰かに対してというよりも、人の行動や会話、集団の空気そのものに対して感じる違和感だ。周囲では自然に受け入れられている振る舞いが、私にはどうしても理解できない、あるいは腑に落ちない、という場面が何度もあった。

これまでの章では、自分自身の体験や考え方について書いてきたが、この章では視点を少し外に向け、**他人の行動を見たときに、私がどのようなズレや違和感を感じてきたのか**について書いていきたい。

それらは単なる好き嫌いではなく、後になって振り返ると、承認欲求や立場意識といった、人の行動原理の違いから生じているように思えるものだった。
