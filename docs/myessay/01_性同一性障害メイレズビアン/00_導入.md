# １．性同一性障害メイレズビアン

私は、47年間、自分のことを普通の男性だと思って生きてきた。性自認や性的指向について深く考えたことはなく、自分が男性であることを疑ったこともなかった。

自分が男性でありながら性自認は女性で、性的指向が女性であることに気が付いたのは、47歳の誕生日を過ぎてから一か月ほど経った頃だった。

男性でありながら、女性として女性を愛する感覚を持っている、という状態である。この状態を毎回説明するのは手間がかかるため、今後は「メイレズビアン」と呼ぶことにする。

このことに気が付いたとき、過去に起きていた不可解な出来事が、一気につながったような感覚があった。これまで感じてきた違和感や噛み合わなさも、「自分はメイレズビアンである」という前提を当てはめることで、なぜそう感じていたのかが理解できるようになった。

この章では、私が過去の経験を振り返りながら、どのような経緯で自分をメイレズビアンだと認識するに至ったのかを、ひとつずつ整理していきたい。

念のために書いておくが、ここで述べる内容は、あくまで現在の私自身の考えであり、科学的な根拠に基づくものではない。サンプル数1の、個人的な視点に過ぎない。
