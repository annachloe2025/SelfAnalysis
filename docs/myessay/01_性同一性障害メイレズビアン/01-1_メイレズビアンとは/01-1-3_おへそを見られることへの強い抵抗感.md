# 1-1-3　おへそを見られることへの強い抵抗感

私は小学校一年生から中学校三年生まで、スイミングスクールに通っていた。喘息気味で体が弱かったためだと聞いているが、その経験のおかげか、それ以降、自分を特別に虚弱だと感じることはあまりなかった。

そのスイミングスクールでのことだが、練習中、私はいつも海水パンツをへその位置まで強く引き上げ、結果的にハイレグのような状態にして、へそを隠していた。自分でも少し不自然な格好だとは感じていたが、それでもへそが露出することに強い抵抗があった。

小学校五年生頃になって、今度はそのハイレグ状態のほうが周囲から見て不自然で、むしろ恥ずかしいのだと気づき、徐々に普通の履き方に戻していった。ただ、その過程でも強い恥ずかしさを感じていた記憶がある。

当時の私は、自分のへそがやや出べそ気味なのだと思い込んでいた。今になって考えれば、週に二回水泳をしていれば体脂肪率が下がり、腹部が引き締まることで、へそが目立って見えるのは特別なことではない。しかし当時は、そのことを理解できず、「見られてはいけないもの」だと感じていた。

現在の視点から振り返ると、この行動は単なる外見上のコンプレックスだけでは説明しきれないようにも思える。メイレズビアンという自己認識を踏まえると、無意識のうちに、自分の身体を「見られる対象」として強く意識し、過度に隠そうとしていた可能性も考えられる。

もちろん、これには明確な根拠があるわけではない。いわゆる性同一性障害は、脳の性と身体の性が一致しない状態だと説明されることが多いが、もしそうだとすれば、当時の私は、本能的に女性的な身体感覚に近い形で、自分の体を守ろうとしていたのかもしれない。この点については、あくまで現在の私なりの仮説に過ぎない。
