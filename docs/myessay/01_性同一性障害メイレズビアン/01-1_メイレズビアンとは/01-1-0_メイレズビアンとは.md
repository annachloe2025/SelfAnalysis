# １-１　メイレズビアンとは

注意点として、メイレズビアンは正式な名称ではなく、俗称です。短い単語でしかも解りやすく性の種類を説明できるのでこの単語を使用します。

メイレズビアンとは、**性自認・性的指向・生物学的性の組み合わせ**を表す用語の一つである。

少し特殊な用語を用いるため、まずそれらの定義を説明する。

**性自認**：
自分自身をどの性別であると認識しているか、という自己認識のこと。

**性的指向**：
恋愛的・性的な関心をどの性別に向けるか、という傾向のこと。

**生物学的性**：
出生時に主に身体的特徴（染色体・生殖器など）によって区分される性別。

**シスジェンダー（cisgender）**：
生物学的性と性自認が一致している人。社会の多数派はこれに該当する。

**トランスジェンダー（transgender）**：
生物学的性と性自認が一致していない人。社会的には少数派である。

**ヘテロセクシャル（heterosexual）**：
異性を恋愛・性的対象とする人。一般に多数派とされる。

**ホモセクシャル（homosexual）**：
同性を恋愛・性的対象とする人。社会的には少数派である。

**メイレズビアン（male lesbian）**：
生物学的性は男性で、性自認は女性（トランスジェンダー）であり、
かつ性的指向が女性に向いている人を指す俗称である。
非常に少数であり、主に当事者的・説明的文脈で用いられる。
