# 1-2 普通ではない自分の考え方

私は子どもの頃から、
「なぜみんなは、こう考えないんだろう」
と思うことが多かった。

当時から、私は自分の考え方が周囲と噛み合っていないという強い違和感を持っていた。
「自分がおかしいのか、それとも周りが考えていないだけなのか」
その判断がつかないまま、ただズレている感覚だけを抱え続けていた。

しかし、自身がメイレズビアンであると気づいてから、この思考の傾向について初めて納得できるようになった。これまで漠然としていた違和感が、「そういう前提で考えていたから噛み合わなかったのだ」と、一つの説明としてつながった感覚があった。

性同一性障害により、脳と身体の性が一致していないことが、自分の思考や欲求の構造に影響しているのではないか。特に、自分にはいわゆる「承認欲求」がほとんど存在しないことが、この独特な考え方の土台になっているのではないか、と考えるようになった。

ここで突然「承認欲求」という言葉が出てきて、唐突に感じるかもしれない。そのため、背景知識として、後の章で**マズローの5段階欲求**について詳しく整理している。可能であれば、そちらを先に読んだ上で、この章を読むと、話のつながりが理解しやすくなると思う。

この章では、そうした前提を踏まえた上で、
**私がどのように他人とは違う考え方をしてきたのか**、
その具体的な感覚やズレについて語っていきたい。
