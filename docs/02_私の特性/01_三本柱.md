# 3つの特性（生得的）

## 「生得的」とは

「生得的（せいとくてき）」とは、**生まれつき持って備わっている特性** のことを指す。後天的（こうてんてき）に学習や環境を通して身についたものではなく、もともとそうなっている、という意味の言葉だ。気質、感受性、性別の認識、知覚の感度といったものがここに含まれる。

ここで挙げる三つの特性は、私が「気づいたときには自分の中にすでにあった」もので、努力や訓練で獲得したわけではないし、後から身についたものでもない。だから生得的、と呼んでいる。

なお、これを「特技」や「強み」として誇りたいわけではない。むしろ、世の中の標準とずれている部分を、ありのままに記述しておくためのラベルだ。

## 第一：性同一性障害（メイレズビアン）

**生物学的男性、性自認女性、性的指向女性。**

47歳のときに気づいた。それまで47年間、自分が他人と何かが違うと感じていた違和感が、この一つの認識で氷解した。

- 子供の頃から走り方を「女の子みたい」と何度も言われていた
- 小学校でハイレグ状態の海水パンツでおへそを隠していた
- レズビアンもののエロ本に最も強く反応した（受け側の女性的視点に没入）
- 格闘ゲームで一貫して女性キャラを選び続けた
- 嫉妬・嫁姑関係・出世競争が「他人事」のように見えた

詳細は [メイレズビアン](02_メイレズビアン.md) を参照。

## 第二：承認欲求がない

**マズローの欲求段階説でいう4段階目の承認欲求が、極めて薄い、あるいはほとんど存在しない。**

私は「他者承認」「自己承認」のどちらの方向にも、承認それ自体を求める動機を持っていない。マズローの承認欲求10項目を一つずつ自己照合した結果、すべてに対して「ほぼ存在しない／弱い／欠けている」という結論になった。

代わりに動いているのは、**合理性と整合性で世界を判定する** モードだ。これを「合理性駆動モード」と呼んでいる。

- 出世したい・偉くなりたい欲求がない
- 肩書・称号への憧れがない
- SNSの「いいね」を「気持ち悪い」と感じる
- マウンティングの動機がないので、知識共有が極端にフラット
- 麻雀で勝ちに執着する人を見ると無意識に手を抜いてしまう

詳細は [承認欲求がない構造](03_承認欲求がない構造.md) と [4段階目薄人間](05_4段階目薄人間.md) を参照。

## 第三：HSP体質

**Highly Sensitive Person。刺激への過敏さと、共感力の強さ。**

私は他者の痛み、物語の登場人物の感情、不当な扱いを目撃したときの動揺、刺激を受けた後の疲労が、平均より明確に強い。これがアニメ・小説への深い没入と、社会の不合理への強い反応を両方生んでいる。

- 中学から国語克服のため読み始めたライトノベルに過剰に没入する習性が始まった
- アニメを「制作者のように」見る癖がある（作画の挙動・カット構成への注意）
- 自分にとって不快なテレビ出演者は無意識にチャンネルを変えてしまう
- 共感力（transportation能力）が高く、物語空間に身体ごと持っていかれる

DNA検査で **「開放性が高い」「情報処理がやや遅い傾向」** が判明している。この組み合わせがHSP的特性と整合する。

詳細は [HSP的感受性](04_HSP的感受性.md) を参照。

## なぜこの三つを基盤として置くのか

私を構成している派生的な特性（功利主義、女性嫌悪、女性キャラへの感情移入、アニメ・小説愛、本質重視、合理性駆動モード、9年×2サイクル仮説、通貨レート違い仮説、自分専用生成システム……）は、すべてこの三つの組み合わせから説明できる、というのが私の現時点での到達点になっている。

例えば：

| 派生特性 | 三つの特性との因果 |
| --- | --- |
| 功利主義・ヒューマニズム・理想主義 | 承認欲求がない × HSP（弱者への共感）× 性同一性障害（社会の本能から距離） |
| アニメ・小説への深い没入 | HSP × 承認欲求がない（世間の評価に依存しない読書） |
| 女性キャラへの感情移入 | 性同一性障害（性自認女性） |
| 物事の本質を重視する | 開放性 × HSP × 承認欲求がない（生きる意味を自作する必要） |
| 倉庫の裏で泣いた | 承認欲求がない（合理が通じない世界の認知崩壊） |
| 自分専用ライトノベル生成システム | HSP × 承認欲求がない（「ツボに正直」が貫ける） |

## 気づいた順番

私はこの三つを、次の順番で言語化していった。

1. 高校〜20代：HSPの存在は感覚としてあったが言葉がなかった
2. 30代：HSPという言葉に出会う
3. 40代前半：マズローの承認欲求が自分にないことを薄々自覚
4. 47歳：**性同一性障害（メイレズビアン）の自覚で全てが繋がる**
5. それ以降：「メイレズビアン → 配偶ゴール不成立 → 地位獲得本能不発動 → 承認欲求の薄さ」という因果連鎖を仮説化（[配偶動機-地位獲得本能の連動仮説](../06_仮説と理論/01_配偶動機-地位獲得本能の連動仮説.md)）

最後に来た発見が、最初のふたつを理解する鍵だった、という構造になっている。
