# メイレズビアン

私は **メイレズビアン**、別の言い方をすれば **AMABレズビアン**（Assigned Male At Birth、出生時に男性と割り当てられたが、性自認は女性で、性的指向も女性）だ。

47歳のときにこのことに気づいた。それまでの47年間、私は自分が何者か分からないまま、違和感だけを抱え続けていた。

## 用語

| 用語 | 意味 |
| --- | --- |
| AMAB | Assigned Male At Birth。出生時に男性と割り当てられた人 |
| メイレズビアン | 「中身レズビアンの男性身体」を指す本人の用語 |
| トランスジェンダー | 性自認と出生時性別が一致しない総称 |
| シスジェンダー | 性自認と出生時性別が一致する人 |

私はこの中で「メイレズビアン」「AMABレズビアン」という呼称を好んで使う。「トランスジェンダー」は一般化された総称で、私の固有の状況（性自認女性 × 性的指向女性）を直接指さないからだ。

## 47歳での気づきまで

47歳までこのことに気づかなかった理由は単純で、**性自認女性 × 性的指向女性の組み合わせは、外形的には「男性で女性が好き」と区別がつかない**からだ。

外側から見ると：
- 男性として生まれて、女性が好き
- これは普通のヘテロセクシュアル男性に見える

内側で起きているのは：
- 自分は女性として、女性を愛している
- 男性身体は単に間違って与えられた器

この内側の感覚は、自分が一人称の主観でしか感じ取れないので、外部からの指摘で気づくことは絶対にない。私は47年間、自分の感覚を「他のヘテロ男性も同じだろう」と誤解し続けていた。

きっかけは記録していない（後で書き起こす）が、ある時点で「自分は中身がレズビアンだ」と理解した瞬間に、過去のあらゆる違和感が一気に繋がった。

## 身体的傍証

メイレズビアンであることの、身体・行動レベルの傍証を整理する。詳細なエピソードは [身体的傍証](../05_根拠とエピソード/02_身体的傍証.md) に。

### 走り方

子供の頃から「女の子みたいな走り方」と何度も指摘されてきた。小学校から成人後（20歳前後）まで、複数の人から複数回、独立に。

高校3年頃、全力で走っても前に進まない感覚があり、自分で観察したところ骨盤が後傾していて、いわゆる女性的な歩行・走行のフォームになっていた。ロンドンブーツの田村淳の走り方に近いと自己分析している。

これはpoint-light walker研究（光点だけで歩行者の性別が判定できるという研究）と整合する。骨盤の傾き・腰の動きという物理レベルで、私の身体は男性身体だが脳の運動制御が女性型である可能性がある。

### おへそを見られることへの抵抗

小学校5年頃まで、海水パンツをハイレグ状態にしておへそを隠していた。男性の通常の海水パンツでは下腹部が露出するが、それを「見られる」ことに強い抵抗があった。

これは女性が水着で身体を見られる感覚に近い。男性の身体感覚としては不自然な強度だ。

### 他人に素肌を触られることへの不快感

身体測定で同級生が触られ遊びをするとき、私は本気で怒っていた。中学では銭湯・温泉が苦手で、今も苦手だ。これも「身体を触られる対象」としての感覚の過敏さで、女性的身体感覚と整合する。

### 性的嗜好

高校時代、エロ本ではレズビアンものに最も強く反応した。特に女性同士の対戦・イカセ合いの構図に。

最初は「女が二人だから興奮が二倍」と単純に思っていたが、後に「自分は受け側の女性的な視点に没入していた」と気づいた。「見る側」より「感じる側」に同一化している。これは男性的なゲイズ（凝視・対象化）の構造ではない。

### 嫉妬の不在

私は嫉妬の感情がほぼ理解できない。嫁姑関係、女性同士の見えない攻撃、フェミニズムの「他の女性に勝ちたい」構造、いずれも自分には実感がない。

これは「承認欲求の不在」と組み合わさることで、私の中で「女性嫌悪」を生んだ時期があった。詳細は[派生的特性のマップ](06_派生的特性のマップ.md) と[認知的傍証](../05_根拠とエピソード/03_認知的傍証.md)を参照。

## 自覚後の状態

身体的移行（ホルモン、手術）は現時点で予定していない。社会的カミングアウトも特にしていない。

理由：
- 私の生活はすでに人間関係比重を最小化したものになっている（朝の新聞配達、Uber Eats）。社会的承認を得たい欲求がないので、カミングアウトの機能的必要性がない
- 身体的移行のコスト（金銭・時間・健康リスク）に対して、得られるリターン（自分が女性身体になる）は、47歳という年齢では費用対効果が悪い
- メイレズビアンであることを自分が知っているだけで、認知の整合性は完成している

ただし、創作活動では明確にメイレズビアンを主人公にしたファンタジー小説を構築している（[ファンタジー創作プロジェクト](../07_作品と趣味/04_ファンタジー創作プロジェクト.md)）。AMABレズビアン主人公が転生先で生物学的女性のレズビアンになる設定で、私のツボの中核を直撃する世界観だ。
