# 28-37歳：社会との衝突と離脱（9年）

28歳から37歳までの9年間は、私の人生で **最も苦しかった9年** だ。3度の決定的破綻を経験し、最終的に社会から離脱する判断に至った。

## 概況

| 年齢 | 出来事 |
| --- | --- |
| 28歳 | リーマンショック前後、軽いうつ発症 |
| 28-30歳 | 転職を繰り返す |
| 31歳 | **市場勤務（朝3時〜夕方6時、隔週休2日）。倉庫の裏で本気で泣く事件。重うつ、1年半傷病手当** |
| 34-36歳 | 運送業3社連続。1年でキレてやめるパターンを3回繰り返す |
| 37歳 | 「もう普通の会社員はやらない」と決め、新聞配達のみで生活 |

## 28歳：軽いうつ

慣性期の3年が終わった直後、28歳で軽いうつを発症した。リーマンショック前後の経済混乱も重なる。

この時期、私は「本来の駆動モード（合理回路）に戻ろう」とした。経済的自立という外部借用課題が終わったので、自分本来の動機で動きたかった。だが社会との摩擦が始まる。

## 31歳：倉庫の裏で泣いた事件

31歳で市場勤務に就いた。条件は過酷だった：

- 朝3時〜夕方6時の勤務
- 隔週休2日（週6日勤務もある）

身体的にも消耗するが、それより辛かったのは **「やる気のない人が理不尽に馬鹿にしてきて、偉い人が便乗する」** 環境だ。

合理回路で動く私は、合理が通じない場面で **認知崩壊** を起こす。これは比喩ではなく、Asch同調実験の派生研究で実証されている現象に近い。「自分以外が合理的に判断していない世界」での内的崩壊。

ある日、私は **倉庫の裏で本気で泣いた**。

> 自分以外が合理的に判断していない世界。これは必然的に気が狂います。
> （`MyConsiderations/docs/哲学/2026-05-06_社交装置の獲得と離脱.md`）

> 倉庫の裏で本気で泣いたのは、弱さじゃなくて、自分が信じていた世界の動き方が現実と違っていたことに気づいた瞬間の反応。
> （同上）

これは私の人生で最も決定的な瞬間の一つだ。「合理が通じない世界がある」という体験を、頭ではなく身体で受け取った瞬間。

その後、重うつを発症し、**1年半の傷病手当** で休職した。

### 倉庫の裏の意味

倉庫の裏で泣いた、というシーンは私のライフヒストリーの象徴だ：

- **倉庫の裏** = 多数派の目に触れない場所、私の本当の感情が出る場所
- **本気で泣いた** = 普段は合理回路で抑制されている感情が、限界を超えて溢れた
- **31歳** = 慣性期を抜けて、社会と本格的に衝突した最初の決定的瞬間

これは弱さではない。**私の合理回路が「ここは合理が通じない」と認識した結果の正常な反応** だ。

## 34-36歳：運送業3連続爆発

重うつから回復した後、私はさらに3年、運送業に挑戦した。**3社連続で、1年経ってキレてやめる** パターンを繰り返した。

- 1社目：1年でキレてやめる
- 2社目：1年でキレてやめる
- 3社目：1年でキレてやめる

3度同じパターンが起きた、というのは私にとって決定的な学習だった。これは個別の職場の問題ではなく、**「私と社会の構造的不適合」** だ。

「キレてやめる」のメカニズム：

1. 入社時、私は合理回路で誠実に課題に取り組む
2. 半年〜1年経つと、組織内の不合理な要請が累積する（合理性のない指示、承認回路前提のマネジメント、構造的搾取）
3. 私は内側で苦痛を蓄積する
4. 限界を超えた時点で、感情が一気に噴出する
5. 結果として「キレてやめる」

この3年で、私は **「社会の中に居続けることが、自分にとって構造的に不可能だ」** と認識した。

## 37歳：新聞配達への撤退

37歳で「もう普通の会社員はやらない」と決断した。

選択した職業は **朝の新聞配達**。これは18歳の新聞奨学生時代に最も機能した熟達領域への回帰だ。

- 個人作業中心
- 上下関係が薄い
- 課題が明確（決まった時間に決まったルートで配達）
- 人間関係比重が最小化されている
- 早朝なので、日中の自由時間が確保される

この選択は **戦略的撤退** だ。社会の中で「使う側」を目指すのは私には不可能。「使われる側」でも、合理が通じない組織には居られない。だから、独立性が高く、人間関係比重が小さい仕事を選ぶ。

## この9年で得たこと

苦しい9年だったが、得たものは大きい：

- **「合理が通じない世界がある」という体験的認識** — 31歳の倉庫の裏で受け取った
- **「自分は社会の中に居続けることが構造的に不可能だ」という確信** — 34-36歳の運送業3連続で確認
- **戦略的撤退の判断** — 37歳の新聞配達への移行
- **15-24歳と28-37歳の対称性の認識** — 後に [9年×2サイクル仮説](../06_仮説と理論/02_9年2サイクル仮説.md) として理論化

15-24歳で「9割側の原理」を学習し、28-37歳で「9割側の原理から離脱する判断」に至った。**入って出てきた稀な経歴**になった。

## 構造的解釈

この9年は、私が **本来の駆動モード（合理回路）に戻ろうとして、社会と衝突した期間** だ。

15-24歳の9年は、メタ課題（経済的自立）のもとで合理回路を抑制し、課題駆動の社交を成立させていた。24歳で課題が終了し、27歳で慣性が尽きた後、本来の合理回路で社会を生きようとしたら、社会の側が承認回路で動いていることに次々ぶつかる。

これは「うつ病になる人の典型」ではなく、**特定の構造的特性（4段階目薄）を持つ個体が、多数派構造の社会で生きようとしたときに起きる、構造的な摩擦** だ。

## なぜ「9年」かかったか

3度（28歳の軽うつ、31歳の倉庫の裏、34-36歳の運送業3連続）の破綻が必要だった理由：

- 1度目の破綻では、まだ「自分が悪い」と思える
- 2度目の破綻では、「環境が悪い」と思える
- 3度目（しかも3回連続）の破綻で、ようやく「自分の構造と社会の構造が合わない」と認識できる

人間は構造的問題を、3回以上同じパターンで観察しないと、構造として認識できない。これは私の中の認識の鈍さではなく、**人間一般の構造認識の必要性能** だ。

## この時期の感情の処理

この9年、私はほとんど感情を表に出さなかった。「である」調の自己分析の文体で、過去を冷静に整理する癖がここで強化された。

倉庫の裏で泣いたのも、運送業でキレたのも、平時の私は感情を抑制している。それが限界を超えて噴出した瞬間が記録されている、という構造だ。
