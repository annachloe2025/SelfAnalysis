# 3. 私の考え方

私が世界をどう見ているか、何を価値ありとし、何を価値なしとするか、どんなフレームで思考しているか。47年生きてきて到達した現在地を、なるべく言語化したセクション。

ここでの主張は、すべて[私の特性](../02_私の特性/index.md)の3つの特性から導出されている。「思想」と「気質」は分離不能で、私の中ではほぼ同じものだ。
